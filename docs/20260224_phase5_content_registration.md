# Phase 5-1c~f: 신규 콘텐츠 대량 등록 + FoodProperties 제거

## 작업 개요
Phase 5 콘텐츠 확정 단계. 테스트 아이템을 삭제하고 실제 콘텐츠로 교체.
어종 19종, 작물 20종(기존 3종 포함), 광물 11종, 요리 24종 등록.
기존 어종/작물/밥의 FoodProperties 제거 (단독 섭취 불가, 요리만 섭취 가능).

## 작업 전 요청사항
- 김/면 아이템: 등록만 (획득 경로 추후)
- 흑요석 조각: 화로에서 minecraft:obsidian 제련
- cooked_rice FoodProperties 제거 (요리 재료 전용)

## 작업 진행 중 결정된 사항
- JIRA: CHOCO-104
- 7개 커밋으로 분할 진행

## 작업 체크리스트
- [ ] 커밋 1: 테스트 아이템 삭제 + FoodProperties 제거
- [ ] 커밋 2: 어종 19종 등록 + 가중치 낚시 루트 모디파이어
- [ ] 커밋 3: 작물 17종 신규 등록 (총 20종)
- [ ] 커밋 4: 광물 11종 등록 + 월드젠
- [ ] 커밋 5: 요리 24종 레시피 + 김/면 아이템
- [ ] 커밋 6: 도감/가격/상점 최종 정리
- [ ] 커밋 7: 테스트 + 빌드 검증 + 문서 마무리

## 인게임 테스트
- [ ] 낚시: 19종 어종이 가중치에 따라 잡히는지 확인
- [ ] 작물: 20종 작물이 심기/성장/수확 되는지 확인
- [ ] 광물: 11종 광물이 월드젠/채굴 되는지 확인
- [ ] 요리: 24종 레시피가 조리대에서 동작하는지 확인
- [ ] 도감: 모든 아이템이 도감에 등록되는지 확인
- [ ] 상점: 씨앗/재료 구매 동작 확인
- [ ] 등급 게이팅: 장비 등급별 레시피/어종/작물 제한 동작 확인
