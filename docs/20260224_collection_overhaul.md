# 도감 시스템 다듬기 + 칭호 GUI 분리

## 작업 개요
- JIRA: CHOCO-113
- 도감 시스템을 모든 마인크래프트 아이템으로 확장
- 칭호를 별도 GUI로 분리 (K키)
- 보상 탭 신규 구현 (마일스톤 보상 수동 수령)
- 도감대 연출 개선

## 작업 전 요청사항
- 기존 도감 데이터 호환 필수

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보
- 카테고리: 8개 (FISH, CROPS, MINERALS, COOKING, BLOCKS, EQUIPMENT, FOOD, MATERIALS)
- Phase A: 커스텀 아이템 명시 등록, Phase B: 바닐라 아이템 자동 분류
- 칭호 GUI: K키, 도감과 분리
- 보상 탭: 마일스톤 달성 시 칭호+아이템 보상 수동 수령
