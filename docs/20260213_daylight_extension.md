# 낮 시간 연장

## 작업 개요
서버의 낮 시간을 연장하여 플레이어 활동 시간을 늘린다.
- 참고: Wispwood 서버 (낮 3배 연장)
- 로드맵 Phase 1.5-2 항목

## 작업 전 요청사항
- JIRA: CHOCO-82

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보

### 마인크래프트 시간 구조
- 1일 = 24,000 틱 (실시간 20분)
- 0~12,000: 낮 (실시간 10분)
- 12,000~24,000: 밤 (실시간 10분)
- 3배 적용 시: 낮 30분 + 밤 10분 = 총 40분/일

### 구현 방식
`LevelTickEvent.Post`를 사용하여 바닐라가 시간을 1 증가시킨 후, 낮 시간대에서 일부 틱의 증가분을 되돌린다 (counter 기반).

**핵심 로직** (multiplier = 3일 때):
- 낮(0~12000) 동안 매 틱마다 카운터 증가
- `counter % 3 != 0`이면 `dayTime`을 1 감소 (바닐라 증가분 취소)
- 결과: 3틱에 1번만 시간 진행 → 낮 3배 연장
- 밤(12000~24000)에는 개입 없음 → 기존 속도

### 변경 파일
- 신규: `DaylightHandler.kt` — 낮 시간 연장 핸들러
- 수정: `Config.kt` — `DAYTIME_MULTIPLIER` 설정 추가
- 수정: `EstherServerMod.kt` — 핸들러 등록
