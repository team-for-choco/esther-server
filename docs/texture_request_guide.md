# Claude.ai 텍스처 요청 가이드

## 공통 지침 (모든 요청에 복사-붙여넣기)

```
마인크래프트 모드용 아이템 텍스처를 만들어줘.

규격:
- 16x16 픽셀아트
- 투명 배경 (PNG)
- 바닐라 마인크래프트 스타일 (외곽선 있는 픽셀아트, 밝은 하이라이트)
- 테마: 동아시아 (한국 기반 + 일본/중국 요소) 마인크래프트 서버

각 아이템을 개별 16x16 PNG 파일로 만들어줘.
```

## 배치 1: 어종 19종 (아이템 텍스처)

```
아래 생선/해산물 아이템 텍스처를 만들어줘. 마인크래프트 바닐라 생선(대구, 연어)과 비슷한 스타일이되, 각 어종의 특징이 구별되게.

일반 등급 (7종) — 약간 수수한 색감:
1. crucian_carp (붕어) — 황금빛 민물 물고기, 둥근 체형
2. sweetfish (은어) — 은빛 날씬한 민물 물고기
3. mackerel (고등어) — 등이 파란 줄무늬, 배는 은색
4. squid_catch (오징어) — 흰/분홍 오징어, 다리 표현
5. anchovy (멸치) — 아주 작고 은빛인 물고기
6. shrimp (새우) — 분홍/주황 새우, 구부러진 형태
7. clam (조개) — 갈색/베이지 조개 껍데기

고급 등급 (6종) — 좀 더 선명한 색감:
8. salmon_catch (연어) — 주황/분홍 살, 바닐라 연어보다 진한 색
9. sea_bream (도미) — 붉은빛 도는 물고기, 높은 체고
10. eel (장어) — 길고 뱀처럼 생긴 갈색/검은 물고기
11. octopus (문어) — 붉은/보라 문어, 다리 표현
12. hairtail (갈치) — 은색으로 길고 납작한 칼 같은 물고기
13. yellowtail (방어) — 노란 줄이 있는 은색 물고기

희귀 등급 (6종) — 화려하고 고급스러운 색감:
14. bluefin_tuna (참다랑어) — 크고 어두운 파란빛 물고기
15. blowfish (복어) — 둥글고 부풀어 있는, 가시 있는 물고기
16. abalone (전복) — 납작한 타원형 조개, 안쪽이 무지갯빛
17. king_crab (대게) — 붉은 큰 게, 다리 표현
18. sea_urchin (성게) — 검은/보라 가시가 난 둥근 형태
19. sturgeon (철갑상어) — 길고 갑옷 같은 비늘, 회색/갈색
```

## 배치 2: 작물 씨앗 17종 + 수확물 17종 (아이템 텍스처)

> 기존 3종(쌀/고추/시금치)은 커스텀 텍스처 있으므로 제외

```
아래 작물의 씨앗과 수확물 텍스처를 만들어줘.
씨앗은 바닐라 밀 씨앗처럼 작은 알갱이 느낌이되, 각 작물마다 색상을 달리해줘.
수확물은 해당 작물의 실제 모습을 16x16로 표현해줘.

일반 등급 (5종):
1. green_onion (파) — 씨앗: 연두 알갱이 / 수확물: 길고 초록색 대파
2. garlic (마늘) — 씨앗: 흰 알갱이 / 수확물: 흰 마늘 통
3. cabbage (배추) — 씨앗: 연두 알갱이 / 수확물: 초록잎에 흰 줄기 배추
4. soybean (콩) — 씨앗: 노란 알갱이 / 수확물: 연노랑 콩알
5. sesame (참깨) — 씨앗: 갈색 알갱이 / 수확물: 작은 흰/베이지 참깨 알

고급 등급 (6종):
6. ginger (생강) — 씨앗: 황토 알갱이 / 수확물: 울퉁불퉁한 황토색 생강 뿌리
7. perilla (깻잎) — 씨앗: 갈색 알갱이 / 수확물: 진한 초록 넓은 잎
8. lotus_root (연근) — 씨앗: 갈색 알갱이 / 수확물: 구멍 뚫린 베이지 연근 단면
9. shiitake (표고버섯) — 씨앗: 갈색 포자 알갱이 / 수확물: 갈색 갓에 흰 줄기 버섯
10. bamboo_shoot (죽순) — 씨앗: 연두 알갱이 / 수확물: 뾰족한 연노랑 죽순
11. wasabi (와사비) — 씨앗: 초록 알갱이 / 수확물: 짧고 굵은 초록 뿌리

희귀 등급 (6종):
12. ginseng (인삼) — 씨앗: 빨간 알갱이 / 수확물: 사람 모양 베이지 뿌리
13. truffle (송로버섯) — 씨앗: 검은 포자 알갱이 / 수확물: 울퉁불퉁한 검은/갈색 덩어리
14. saffron (사프란) — 씨앗: 주황 알갱이 / 수확물: 진한 빨강/주황 실 모양
15. matsutake (송이버섯) — 씨앗: 흰 포자 알갱이 / 수확물: 굵은 줄기에 갈색 갓 버섯
16. yuzu (유자) — 씨앗: 노란 알갱이 / 수확물: 울퉁불퉁한 노란 감귤
17. green_tea (녹차) — 씨앗: 진초록 알갱이 / 수확물: 작고 진한 초록 찻잎 뭉치
```

## 배치 3: 작물 재배 단계 블록 텍스처 (17종 x 4단계 = 68개)

> 기존 3종(쌀/고추/시금치)은 커스텀 텍스처 있으므로 제외
> 이건 블록 텍스처라서 16x16 크기에 작물이 바닥부터 자라는 모습

```
마인크래프트 작물 재배 블록 텍스처를 만들어줘.
바닐라 밀처럼 바닥에서 위로 자라는 4단계(stage0~3) 블록 텍스처야.
- stage0: 씨앗이 막 싹 틔운 상태 (낮고 작은 새싹)
- stage1: 조금 자란 상태 (줄기 형성)
- stage2: 거의 다 자란 상태 (잎/열매 보임)
- stage3: 완전 성숙 (수확 가능, 가장 크고 화려)

한 작물당 4개의 16x16 PNG가 필요해. 투명 배경이고, 작물만 표현.

아래 17종 작물 각각 stage0~stage3 4장씩 만들어줘:

일반: green_onion(파), garlic(마늘), cabbage(배추), soybean(콩), sesame(참깨)
고급: ginger(생강), perilla(깻잎), lotus_root(연근), shiitake(표고버섯), bamboo_shoot(죽순), wasabi(와사비)
희귀: ginseng(인삼), truffle(송로버섯), saffron(사프란), matsutake(송이버섯), yuzu(유자), green_tea(녹차)
```

## 배치 4: 광물 아이템 텍스처

```
마인크래프트 광물 아이템 텍스처를 만들어줘.
원석(raw)은 바닐라 raw_iron/raw_gold처럼 울퉁불퉁한 덩어리,
주괴(ingot)는 바닐라 iron_ingot처럼 사다리꼴 금속 막대 형태.

일반 등급 (5개):
1. tin_ore_raw (주석 원석) — 은빛에 약간 밝은 회색 덩어리
2. tin_ingot (주석 주괴) — 밝은 은회색 주괴
3. zinc_ore_raw (아연 원석) — 청회색 덩어리
4. zinc_ingot (아연 주괴) — 청회색 주괴
5. jade_raw (옥 원석) — 진한 초록빛 반투명 원석

고급 등급 (6개):
6. silver_ore_raw (은 원석) — 밝은 은색 반짝이는 덩어리
7. silver_ingot (은 주괴) — 반짝이는 은색 주괴
8. ruby_raw (루비 원석) — 진한 붉은색 보석 원석
9. sapphire_raw (사파이어 원석) — 진한 파란색 보석 원석
10. titanium_ore_raw (티타늄 원석) — 어두운 은회색, 금속적인 덩어리
11. titanium_ingot (티타늄 주괴) — 어두운 은색 주괴, 철보다 진함

희귀 등급 (5개):
12. platinum_ore_raw (백금 원석) — 매우 밝은 흰은색 덩어리
13. platinum_ingot (백금 주괴) — 매우 밝은 흰은색 주괴
14. opal_raw (오팔 원석) — 무지갯빛 도는 흰/파스텔 원석
15. tanzanite_raw (탄자나이트 원석) — 보라/파랑 그라데이션 보석 원석
16. obsidian_shard (흑요석 조각) — 검은 유리 같은 뾰족한 조각
```

## 배치 5: 광물 광석 블록 텍스처 (10종 x 2변형 = 20개)

```
- ruby_ore (루비) — 붉은 알갱이

- tanzanite_ore (탄자나이트) — 보라/파랑 알갱이
```

## 배치 7: 장비/도구/블록

### 7-A: 도구 4종 x 등급 3단계 = 12장

```
마인크래프트 도구 아이템 텍스처를 만들어줘.
도구 4종 각각에 대해 등급별 3가지 버전이 필요해.

등급별 아우라 규칙:
- 일반 (common): 아우라 없음. 도구 본체만. 수수한 느낌.
- 고급 (fine): 도구 주변에 초록색 아우라/빛남 효과. 살짝 고급스러운 느낌.
- 희귀 (rare): 도구 주변에 파란색 아우라/빛남 효과. 매우 고급스럽고 신비로운 느낌.

아우라는 도구 외곽을 감싸는 은은한 빛 픽셀로 표현해줘. 너무 과하지 않게.

도구 4종 (각각 일반/고급/희귀 3장씩):

1. special_fishing_rod (특수 낚싯대) — 바닐라 낚싯대와 비슷하되 더 정교한 디자인
   - special_fishing_rod.png (일반, 아우라 없음)
   - special_fishing_rod_fine.png (고급, 초록 아우라)
   - special_fishing_rod_rare.png (희귀, 파란 아우라)

2. special_hoe (특수 호미) — 바닐라 호미와 비슷하되 더 정교한 디자인
   - special_hoe.png (일반)
   - special_hoe_fine.png (고급, 초록 아우라)
   - special_hoe_rare.png (희귀, 파란 아우라)

3. special_pickaxe (특수 곡괭이) — 바닐라 곡괭이와 비슷하되 더 정교한 디자인
   - special_pickaxe.png (일반)
   - special_pickaxe_fine.png (고급, 초록 아우라)
   - special_pickaxe_rare.png (희귀, 파란 아우라)

4. special_cooking_tool (특수 조리도구) — 프라이팬 또는 국자 형태의 조리 도구
   - special_cooking_tool.png (일반)
   - special_cooking_tool_fine.png (고급, 초록 아우라)
   - special_cooking_tool_rare.png (희귀, 파란 아우라)
```

### 7-B: 기타 아이템 + 블록

```
마인크래프트 기타 아이템/블록 텍스처를 만들어줘.

기타 아이템 (4종):
1. enhancement_stone (강화석) — 빛나는 초록/에메랄드 보석 조각
2. land_deed (토지 증서) — 접힌 양피지에 빨간 인장
3. sprayer (분무기) — 유리병에 분무 노즐
4. watering_can (물뿌리개) — 금속 물뿌리개 형태

블록 (3종, 각 16x16 x 필요 면 수):
5. cooking_station (조리대) — 윗면: 도마/칼, 앞면: 서랍, 옆면: 나무 질감
6. collection_pedestal (도감대) — 돌 질감에 금색 장식, 책/두루마리
```
