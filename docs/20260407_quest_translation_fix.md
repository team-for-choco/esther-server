# 퀘스트 번역 중복 키 제거 + 아이템명 불일치 수정

## 작업 개요

ko_kr.json에 퀘스트 번역 블록이 중복 등록되어 구버전 잘못된 번역이 덮어쓰던 문제를 수정.
퀘스트에 표시되는 아이템 이름을 실제 인게임 아이템명과 일치시킴.

CHOCO-142

## 작업 전 요청사항

- 아이템 이름과 일일/주간 퀘스트의 요구 아이템 이름이 맞지 않는 경우 수정
- 퀘스트 번역을 아이템 이름에 맞춰달라는 요청

## 작업 진행 중 결정된 사항

- ko_kr.json 내 퀘스트 번역 블록이 두 군데 중복 등록되어 있었음
- JSON 특성상 마지막 값이 적용되어 구버전 잘못된 번역이 실제로 표시되고 있었음
- 구버전 중복 블록 전체 제거 후 올바른 번역만 유지
