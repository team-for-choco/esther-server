# 마을 귀환서 구현

## 작업 개요
사용 시 거점(오버월드) 스폰지점으로 텔레포트하는 소모품 아이템 구현.

## 작업 전 요청사항
- 사용 후 3초 카운트다운 (3→2→1 화면 표시) 후 텔레포트
- Common 등급, 64스택
- 야생 차원에서도 사용 가능
- 획득 방법: 추후 결정
