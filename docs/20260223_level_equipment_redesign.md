# 전문 레벨/장비 효과 재설계

## 작업 개요
기존 CHOCO-100에서 구현한 전문 레벨/장비 강화 효과를 전면 재설계한다.
- **전문 레벨**: 마일스톤 방식 → 점진적 성장 + 보관함 확장 마일스톤
- **장비 강화**: 수치 보너스 → 도구 자체가 특별해지는 방향
- **신규**: 전문 보관함 시스템, 요리 시간 시스템, 물뿌리개 아이템

## 작업 전 요청사항
- CHOCO-100 (기존 레벨/장비 효과) 구현 완료 상태
- 기존 효과를 제거하고 새 설계로 교체

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보

### 설계 철학
- **전문 레벨** = "경험의 보상" — 반복 플레이로 자연스럽게 성장, 숙련도 기반 보너스
- **장비 강화** = "투자의 보상" — 재화/재료를 소모해서 도구 자체를 특별하게 만듦

---

### 전문 레벨 — 점진적 보너스 (Lv1~50)

레벨이 오를 때마다 소폭 보너스가 누적된다.

| 분야 | 효과 | 레벨당 | Lv50 최대 |
|------|------|--------|-----------|
| 채광 | 채굴 속도 증가 | +1% | +50% (1.5배) |
| 낚시 | Lure 대기시간 감소 | -1틱 | -50틱 (2.5초) |
| 농사 | 작물 성장 시간 단축 | -1% | -50% (절반) |
| 요리 | 요리 시간 단축 | -0.05초 | 2.5초 (기본 5초) |

### 전문 레벨 — 마일스톤 (보관함 확장)

| 레벨 | 효과 |
|------|------|
| Lv10 | 전문 보관함 해금 (5칸) |
| Lv20 | +5칸 (총 10칸) |
| Lv30 | +5칸 (총 15칸) |
| Lv40 | +5칸 (총 20칸) |
| Lv50 | +5칸 (총 25칸) |

4개 분야 모두 동일한 구조. 추후 분야별 특수 해금 마일스톤 추가 예정 (별도 작업).

### 전문 보관함 시스템

- **접근**: 별도 키 또는 명령어로 전용 GUI 오픈
- **탭 구성**: 일반 인벤토리 / 채광 / 낚시 / 농사 / 요리 (총 5탭)
- **일반 탭**: 바닐라 인벤토리 슬롯을 그대로 표시
- **분야 탭**: 해당 분야 전용 보관함 슬롯 표시
- **하단**: 핫바 항상 표시
- **아이템 이동**: 탭 간 자유 이동 (Shift+클릭 포함)
- **자동 분류**: 아이템 획득 시 해당 분야 보관함 우선 → 꽉 차면 일반 인벤토리
- **슬롯 제한**: 해당 분야 아이템만 넣을 수 있음 (`mayPlace()` 필터링)
- **저장**: 플레이어별 SavedData로 NBT 저장

### 장비 강화 효과 (Lv0~5)

#### 곡괭이

| 강화 단계 | 채굴 등급 | 특수 효과 |
|-----------|-----------|-----------|
| Lv0 | 나무 | - |
| Lv1 | 돌 | - |
| Lv2 | 철 | - |
| Lv3 | 다이아 | - |
| Lv4 | 다이아 | 희귀 강화석 드롭 |
| Lv5 | 다이아 | 희귀 강화석 드롭 + 광맥 감지 |

효과는 누적. 이후 확장(Lv6+)에서 행운, 범위 채굴 등 추가 예정.

##### 희귀 강화석 드롭 확률 (채광)

| 광물 등급 | 확률 |
|-----------|------|
| 일반 (석탄, 철 등) | 0.5% |
| 고급 (금, 다이아 등) | 1% |
| 희귀 (커스텀 고급) | 2% |

#### 낚싯대

| 강화 단계 | 효과 |
|-----------|------|
| Lv0 | 기본 (커스텀 어종 X) |
| Lv1 | 일반 등급 어종 해금 |
| Lv2 | 고급 등급 어종 해금 |
| Lv3 | 2연낚 5% |
| Lv4 | 희귀 강화석 낚시 |
| Lv5 | 비 없이 보너스 |

효과는 누적. 희귀 등급 어종은 이후 확장(Lv6+)에서 해금 예정.

##### 희귀 강화석 드롭 확률 (낚시)

| 확률 |
|------|
| 1% (일괄) |

#### 호미

| 강화 단계 | 효과 |
|-----------|------|
| Lv0 | 기본 (커스텀 작물 X) |
| Lv1 | 일반 등급 작물 해금 |
| Lv2 | 고급 등급 작물 해금 |
| Lv3 | 물뿌리개 사용 가능 (6회 충전) |
| Lv4 | 5% 확률 다수확 (2개) |
| Lv5 | 희귀 등급 작물 해금 |

효과는 누적. 강화석 드롭 없음 (채광/낚시에서만 획득).

##### 물뿌리개 (신규 아이템)
- 분무기의 상위 도구
- 최대 6회 충전 (분무기는 3회)
- 호미 Lv3 이상에서 사용 가능

#### 요리 도구

| 강화 단계 | 효과 |
|-----------|------|
| Lv0 | 일반 등급 레시피 해금 |
| Lv1 | 고급 등급 레시피 해금 |
| Lv2 | 동시 조리대 2개 |
| Lv3 | 동시 조리대 4개 |
| Lv4 | 5% 확률 요리 2개 |
| Lv5 | 희귀 등급 레시피 해금 |

효과는 누적. 기본 레시피 없음 — Lv0부터 커스텀 레시피만 사용.

---

### 요리 시간 시스템 (신규)

- **기존**: 재료 투입 + 조리 도구 상호작용 → 즉시 완성
- **변경**: 재료 투입 + 조리 도구 상호작용 → **5초 대기** → 완성
- 전문 레벨이 오르면 5초가 점점 줄어듦 (Lv50에서 2.5초)
- 플레이어가 자리를 떠나도 조리대가 알아서 완성
- 대기 중 사운드 + 파티클로 시각적 피드백 제공
