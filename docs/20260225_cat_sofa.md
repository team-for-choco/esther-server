# 고양이 소파 가구 (2x2x2 멀티블록 + 앉기)

## 작업 개요
뽑기권 시스템에 들어갈 가구 아이템의 첫 테스트로, 2x2x2 멀티블록 고양이 소파를 구현한다.
단색 텍스처 7장 + 블록 기하학으로 고양이 형태를 표현하고, 기존 SeatEntity를 재활용하여 앉기 기능까지 포함한다.

CHOCO-118

## 작업 전 요청사항
- 기존 SeatEntity, QuestBoard 멀티블록 패턴 재활용

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보

### 텍스처 접근 방식 변경
- **초기**: AI 생성 32x32 상세 텍스처 19장 → 텍스처 방향 매핑 어려움, 인게임에서 아쉬운 결과
- **최종**: 단색 16x16 텍스처 7장 + 블록 엘리먼트 기하학으로 고양이 형태 표현
  - wood (#8B6914), wood_top (#A07828), wood_floor (#5C4A1E)
  - cushion (#F5E6C8), cat (#E8923A), eye (#FFFFFF), nose (#FFB6C1)
  - 고양이 눈/코/귀 등은 작은 엘리먼트 블록으로 색상 차이 표현

### 블록 구조 (facing=SOUTH 기준, 앞에서 본 모습)
```
위층 (dy=1):
  [등받이-좌석] [고양이뒤+등받이]     ← back (dz=1)
  [쿠션-좌석]   [고양이앞+얼굴]      ← front (dz=0)
   seat(dx=0)   cat(dx=1)

아래층 (dy=0):
  [프레임]      [프레임]              ← back
  [프레임★]     [프레임]              ← front  (★=master)
   seat(dx=0)   cat(dx=1)
```

총 8블록: master 1 + dummy 7 (PART 0~6)

### 파트 인덱스 매핑
| 인덱스 | 위치 | 역할 | 모델 |
|--------|------|------|------|
| master | (0,0,0) | 프레임-앞-좌석 | cat_sofa.json |
| part 0 | (dx=1,0,0) | 프레임-앞-고양이 | cat_sofa_part0.json |
| part 1 | (0,0,dz=1) | 프레임-뒤-좌석 | cat_sofa_part1.json |
| part 2 | (dx=1,0,dz=1) | 프레임-뒤-고양이 | cat_sofa_part2.json |
| part 3 | (0,dy=1,0) | 쿠션-좌석 (앉기) | cat_sofa_part3.json |
| part 4 | (dx=1,dy=1,0) | 고양이-앞+얼굴 | cat_sofa_part4.json |
| part 5 | (0,dy=1,dz=1) | 등받이-좌석 (앉기) | cat_sofa_part5.json |
| part 6 | (dx=1,dy=1,dz=1) | 고양이뒤+등받이 | cat_sofa_part6.json |

### 앉기 파트
- part 3 (쿠션-좌석) 또는 part 5 (등받이-좌석) 우클릭 → 앉기
- 좌석 위치: master 블록 기준 dx=0, dy=1 (쿠션 블록 중앙, Y offset 0.3)

### 멀티블록 파괴 로직 (CHOCO-117 게시판에도 동일 적용)
- **핵심 수정**: `level.destroyBlock()`은 `playerWillDestroy()`를 트리거하지 않음
- 더미 블록 파괴 시: 마스터 위치/facing 정보로 전체 위치 계산 → 모든 더미+마스터 제거 + 아이템 드롭
- 마스터 블록 파괴 시: 기존대로 `playerWillDestroy()`에서 모든 더미 제거
- 게시판도 동일 패턴으로 수정 + 경도 -1.0→2.5 변경 + 루트테이블 추가

### 인벤토리 아이템 모델
- 8블록 전체를 16×16×16으로 축소한 미니어처 3D 모델 (cat_sofa_item.json)
- 프레임 다리+상판, 쿠션, 등받이, 고양이(몸통+머리+귀+꼬리+눈+코) 16개 엘리먼트
