# 고양이 소파 가구 (2x2x2 멀티블록 + 앉기)

## 작업 개요
뽑기권 시스템에 들어갈 가구 아이템의 첫 테스트로, 2x2x2 멀티블록 고양이 소파를 구현한다.
AI로 생성한 32x32 텍스처 19장을 적용하고, 기존 SeatEntity를 재활용하여 앉기 기능까지 포함한다.

## 작업 전 요청사항
- AI 텍스처 19장 준비 완료 (`docs/data/files (4)/`)
- 기존 SeatEntity, QuestBoard 멀티블록 패턴 재활용

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보

### 블록 구조 (facing=NORTH, 앞에서 본 기준)
```
위층 (dy=1):
  [등받이-좌석] [고양이뒤+등받이]     ← back (dz=1)
  [쿠션-좌석]   [고양이앞]            ← front (dz=0)
   seat(dx=0)   cat(dx=1)

아래층 (dy=0):
  [프레임]      [프레임]              ← back
  [프레임★]     [프레임]              ← front  (★=master)
   seat(dx=0)   cat(dx=1)
```

총 8블록: master 1 + dummy 7 (PART 0~6)

### 파트 인덱스 매핑
| 인덱스 | 위치 | 역할 |
|--------|------|------|
| master | (0,0,0) | 프레임-앞-좌석 |
| part 0 | (dx=1,0,0) | 프레임-앞-고양이 |
| part 1 | (0,0,dz=1) | 프레임-뒤-좌석 |
| part 2 | (dx=1,0,dz=1) | 프레임-뒤-고양이 |
| part 3 | (0,dy=1,0) | 쿠션-좌석 (앉기) |
| part 4 | (dx=1,dy=1,0) | 고양이-앞 |
| part 5 | (0,dy=1,dz=1) | 등받이-좌석 (앉기) |
| part 6 | (dx=1,dy=1,dz=1) | 고양이뒤+등받이 |

### 앉기 파트
- part 3 (쿠션-좌석) 또는 part 5 (등받이-좌석) 우클릭 → 앉기
- 좌석 위치: master 블록 기준 dx=0, dy=1 (쿠션 블록 중앙, Y offset 0.3)
