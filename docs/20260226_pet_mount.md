# 펫(탈것) 시스템 테스트 구현

## 작업 개요
펫 보관함 GUI에서 펫을 선택하면 플레이어가 탑승한 상태로 소환되는 탈것 시스템.
등급에 따라 속도가 달라지며, 테스트로 일반(Common) 등급 고양이 펫 1종만 우선 구현.
코드 기반 엔티티 모델 사용 (추후 GeckoLib 등으로 교체 가능).

CHOCO-119

## 작업 전 요청사항
- 고양이 모델: 코드 기반 사족보행 모델 (몸통 + 머리 + 다리 4개 + 꼬리)
- 텍스처: 플레이스홀더 16x16 (추후 교체)
- 속도: 스프린트(0.13)보다 ~8% 빠른 0.14

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보
- PetEntity는 PathfinderMob 기반 (탑승 조종 API 활용)
- 하차 시 자동 디스폰, 저장 안 됨
- H키로 보관함 열기
- 플레이어에게 기본 CAT_COMMON 1개 제공
