# 일반 사용자 명령어 목록

## 작업 개요

모드에서 일반 사용자(권한 레벨 0)가 사용할 수 있는 명령어를 정리한 문서.
관리자 전용 명령어(`hasPermission(2)`)는 별도 섹션에 기재.

---

## 일반 사용자 명령어

### `/송금` — 화폐 (`MoneyCommand.kt`)

| 명령어 | 설명 |
|--------|------|
| `/송금 <플레이어> <금액>` | 온라인 플레이어에게 송금 |

---

### `/토지` — 청크 클레임 (`ClaimCommand.kt`)

| 명령어 | 설명 |
|--------|------|
| `/토지 정보` | 현재 서있는 청크의 클레임 정보 (소유자, 권한, 신뢰 인원) |
| `/토지 목록` | 내가 클레임한 청크 전체 목록 |
| `/토지 제거` | 현재 청크 클레임 해제 (본인 소유 청크만) |
| `/토지 세팅` | 현재 청크 권한 설정 확인 |
| `/토지 세팅 파괴 허용\|금지` | 타인의 블록 파괴 허용/금지 |
| `/토지 세팅 설치 허용\|금지` | 타인의 블록 설치 허용/금지 |
| `/토지 세팅 상호작용 허용\|금지` | 타인의 상호작용 허용/금지 |
| `/토지 플레이어 추가 <플레이어>` | 신뢰 플레이어 추가 (내 모든 클레임에 적용) |
| `/토지 플레이어 제거 <플레이어>` | 신뢰 플레이어 제거 |
| `/토지 플레이어 목록` | 신뢰 플레이어 목록 |

---

### `/전문기술` — 전문 레벨 (`ProfessionCommand.kt`)

| 명령어 | 설명 |
|--------|------|
| `/전문기술` | 내 전문 분야별 레벨 및 XP 현황 확인 |

---

### `/칭호` — 칭호 (`TitleCommand.kt`)

| 명령어 | 설명 |
|--------|------|
| `/칭호 목록` | 칭호 전체 목록 (해금/잠금 상태, 진행도 포함) |
| `/칭호 선택 <id>` | 칭호 장착 (해금된 칭호만 가능, 탭 자동완성 지원) |
| `/칭호 해제` | 장착 중인 칭호 해제 |

---

### `/퀘스트` — 퀘스트 (`QuestCommand.kt`)

| 명령어 | 설명 |
|--------|------|
| `/퀘스트 정보` | 일일/주간 퀘스트 진행 현황 및 수령 상태 |

---

## 관리자 전용 명령어 (`hasPermission(2)`)

### `/money admin`
| 명령어 | 설명 |
|--------|------|
| `/money admin add <플레이어> <금액>` | 플레이어 잔액 증가 |
| `/money admin remove <플레이어> <금액>` | 플레이어 잔액 감소 |
| `/money admin set <플레이어> <금액>` | 플레이어 잔액 직접 설정 |

### `/토지 admin`
| 명령어 | 설명 |
|--------|------|
| `/토지 admin fakeclaim` | 현재 청크를 FakePlayer 소유로 설정 |
| `/토지 admin forceremove` | 현재 청크 클레임 강제 해제 |

### `/전문기술 admin`
| 명령어 | 설명 |
|--------|------|
| `/전문기술 admin setlevel <플레이어> <전문> <레벨>` | 플레이어 전문 레벨 직접 설정 |
| `/전문기술 admin addxp <플레이어> <전문> <xp>` | 플레이어 전문 XP 추가 |

### `/shop`
| 명령어 | 설명 |
|--------|------|
| `/shop summon <type>` | 지정 유형 상인 NPC 소환 |
| `/shop remove` | 가장 가까운 상인 NPC 제거 |
| `/shop remove all` | 현재 레벨의 상인 NPC 전체 제거 |

### `/wild`
| 명령어 | 설명 |
|--------|------|
| `/wild reset` | 야생 차원 초기화 (플레이어 대피 후 차원 파일 삭제) |

### `/퀘스트 admin`
| 명령어 | 설명 |
|--------|------|
| `/퀘스트 admin reset daily [플레이어]` | 일일 퀘스트 리셋 |
| `/퀘스트 admin reset weekly [플레이어]` | 주간 퀘스트 리셋 |
| `/퀘스트 admin reset all [플레이어]` | 일일+주간 퀘스트 전체 리셋 |
