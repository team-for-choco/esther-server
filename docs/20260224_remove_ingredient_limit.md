# Phase 5-1b2: 조리대 재료 개수 제한 제거

## 작업 개요
기존 `MAX_INGREDIENTS = 4` 제한을 제거하여 조리대에 무제한 재료 투입 가능하도록 변경.
레시피는 비공개하고 유저가 직접 조합을 찾게 하므로, 재료 개수 제한이 불필요.

## 작업 전 요청사항
- CHOCO-101 (품질 시스템 제거) 완료 후 진행

## 작업관련 주요 정보
- `CookingStationBlock.kt`의 `MAX_INGREDIENTS = 4` 상수 및 체크 로직 제거
- 조리대 가득 참 메시지(`cooking_station_full`) 불필요 → 제거
- 레시피 매칭은 기존대로 작동 (재료가 맞지 않으면 실패)
