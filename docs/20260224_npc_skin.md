# NPC 외형 변경: 빌리저 → 플레이어 스킨 모델

## 작업 개요
NPC 상인의 외형을 바닐라 빌리저 모델에서 플레이어 스킨 모델(PlayerModel)로 변경.
상인 종류별(5종)로 각각 다른 커스텀 스킨을 적용할 수 있도록 구조 변경.

**JIRA**: CHOCO-114

## 작업 전 요청사항
- 없음

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보

### 변경 파일
| 파일 | 변경 |
|------|------|
| `MerchantEntity.kt` | SynchedEntityData로 merchantType 클라이언트 동기화 |
| `MerchantEntityRenderer.kt` | 전면 재작성: VillagerModel → PlayerModel |
| `EstherServerMod.kt` | Entity 크기 0.6×1.95 → 0.6×1.8 |
| `textures/entity/merchant/*.png` ×5 | 신규 — 상인별 플레이스홀더 스킨 |

### 스킨 규격 (커스텀 스킨 제작 시 참고)
- **크기**: 64×64 픽셀
- **형식**: PNG, 투명 배경 지원
- **모델**: WIDE (Steve형, 팔 너비 4px)
- **레이아웃**: 마인크래프트 표준 플레이어 스킨 레이아웃
  - 상단 32px: 머리(+오버레이) + 몸통 + 오른팔 + 오른다리
  - 하단 32px: 왼다리(+오버레이) + 몸통 오버레이 + 왼팔(+오버레이)
- **참고**: https://minecraft.wiki/w/Skin

### 상인별 텍스처 경로
| 상인 종류 | 텍스처 파일 |
|-----------|------------|
| 씨앗 상인 | `assets/estherserver/textures/entity/merchant/seeds.png` |
| 음식 상인 | `assets/estherserver/textures/entity/merchant/food.png` |
| 광물 상인 | `assets/estherserver/textures/entity/merchant/minerals.png` |
| 특수 상인 | `assets/estherserver/textures/entity/merchant/special.png` |
| 대장장이 | `assets/estherserver/textures/entity/merchant/blacksmith.png` |

### 스킨 교체 방법
1. 64×64 PNG 스킨 파일을 준비
2. 위 텍스처 경로에 해당하는 파일을 교체
3. `./gradlew build`로 빌드
4. 모드 JAR 배포

### 커스텀 스킨 제작 프롬프트

AI에게 스킨을 요청할 때 **공통 규격 프롬프트 + 개별 프롬프트**를 합쳐서 사용한다.

#### 공통 규격 프롬프트

```
마인크래프트 플레이어 스킨을 제작해줘.

【필수 규격】
- 크기: 정확히 64×64 픽셀
- 형식: PNG, 투명 배경 (사용하지 않는 영역은 완전 투명)
- 모델: WIDE (Steve형, 팔 너비 4px)
- 레이아웃: 마인크래프트 표준 플레이어 스킨 레이아웃을 정확히 따를 것
  - 좌표 (x, y, w, h):
    - 머리:      (0,0,32,16) 영역 — 앞(8,8,8,8) 뒤(24,8,8,8) 좌(0,8,8,8) 우(16,8,8,8) 위(8,0,8,8) 아래(16,0,8,8)
    - 몸통:      (16,16,24,16) 영역 — 앞(20,20,8,12) 뒤(32,20,8,12) 좌(16,20,4,12) 우(28,20,4,12)
    - 오른팔:    (40,16,16,16) 영역 — 앞(44,20,4,12) 뒤(52,20,4,12) 좌(40,20,4,12) 우(48,20,4,12)
    - 오른다리:  (0,16,16,16) 영역 — 앞(4,20,4,12) 뒤(12,20,4,12) 좌(0,20,4,12) 우(8,20,4,12)
    - 왼다리:    (16,48,16,16) 영역
    - 왼팔:      (32,48,16,16) 영역
    - 오버레이(머리): (32,0,32,16) 영역
    - 오버레이(몸통): (16,32,24,16) 영역
    - 오버레이(오른팔): (40,32,16,16) 영역
    - 오버레이(오른다리): (0,32,16,16) 영역
    - 오버레이(왼다리): (0,48,16,16) 영역
    - 오버레이(왼팔): (48,48,16,16) 영역
    
    【중요: 머리 텍스처 주의사항】
  - 눈과 얼굴은 반드시 머리 앞면 (8,8)-(15,15) 영역에만 그릴 것
  - 머리 오른쪽 옆면 (0,8)-(7,15): 귀/머리카락 옆면만. 눈 절대 금지
  - 머리 왼쪽 옆면 (16,8)-(23,15): 귀/머리카락 옆면만. 눈 절대 금지
  - 머리 뒷면 (24,8)-(31,15): 뒷머리만. 눈 절대 금지
  - 옆면에서 보았을 때 피부색+머리카락만 보여야 한다

【스타일 가이드】
- 마인크래프트 특유의 픽셀아트 스타일
- 깔끔하고 알아보기 쉬운 디자인
- 서버의 아기자기한 분위기에 어울리는 친근한 느낌
```

#### 1. 씨앗 상인 (seeds.png)

```
【캐릭터 콘셉트: 씨앗 상인】
- 직업: 씨앗과 농작물을 판매하는 농부 상인
- 복장: 밀짚모자, 녹색 앞치마, 갈색 작업복, 흙 묻은 장화
- 색상 톤: 녹색 + 갈색 + 베이지 (자연/농업 느낌)
- 특징: 따뜻하고 소박한 농부 이미지, 주머니에 씨앗이 들어있는 느낌
```

#### 2. 음식 상인 (food.png)

```
【캐릭터 콘셉트: 음식 상인】
- 직업: 요리 재료와 음식을 판매하는 요리사 상인
- 복장: 하얀 요리사 모자(토크), 흰색 요리사 복, 앞치마
- 색상 톤: 흰색 + 빨간색 포인트 (요리사/식당 느낌)
- 특징: 깔끔한 요리사 이미지, 앞치마에 음식 얼룩이 약간 있는 디테일
```

#### 3. 광물 상인 (minerals.png)

```
【캐릭터 콘셉트: 광물 상인】
- 직업: 광물과 원석을 판매하는 광부 상인
- 복장: 광부 헬멧(헤드램프 달린), 두꺼운 작업복, 가죽 장갑, 튼튼한 부츠
- 색상 톤: 회색 + 진한 갈색 + 노란색 포인트 (광산/금속 느낌)
- 특징: 얼굴에 석탄 그을음, 강인하고 묵직한 이미지
```

#### 4. 특수 상인 (special.png)

```
【캐릭터 콘셉트: 특수 상인】
- 직업: 희귀한 특수 아이템을 판매하는 신비로운 상인
- 복장: 보라색/남색 로브, 별 무늬 장식, 두건 또는 후드
- 색상 톤: 보라색 + 남색 + 금색 포인트 (신비/마법 느낌)
- 특징: 신비롭고 미스터리한 분위기, 로브 자락에 별/달 무늬
```

#### 5. 대장장이 (blacksmith.png)

```
【캐릭터 콘셉트: 대장장이】
- 직업: 장비 강화와 도구를 판매하는 대장장이 상인
- 복장: 가죽 앞치마, 두꺼운 팔 보호대, 민소매 또는 걷어올린 소매, 장화
- 색상 톤: 검은색 + 진한 빨간색 + 주황색 (대장간/불꽃 느낌)
- 특징: 근육질 체형, 얼굴에 그을음, 강인하고 듬직한 이미지
```
