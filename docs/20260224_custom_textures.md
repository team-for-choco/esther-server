### 도구 등급별 모델 분기
- 강화 레벨 → 등급 매핑: Lv0-2=일반, Lv3-4=고급, Lv5=희귀
- 등급 색상: 일반=아우라 없음, 고급=초록색 아우라, 희귀=파란색 아우라
- 구현: `EnhancementHandler.syncCustomModelData()` → `CustomModelData(floats)` 설정
- 아이템 정의: `range_dispatch` + `custom_model_data` 프로퍼티로 모델 선택
- 대상 도구: 특수 낚싯대, 특수 괭이, 특수 곡괭이, 특수 조리도구

### 모델 JSON 수정
- 블록 모델: 바닐라 텍스처 참조 → 커스텀 텍스처 참조로 변경 (~90개)
- 아이템 모델: 바닐라 텍스처 참조 → 커스텀 텍스처 참조로 변경 (~60개)
- 신규 모델: 도구 등급별 `_fine`, `_rare` 변형 8개
