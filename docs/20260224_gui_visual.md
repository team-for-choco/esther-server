## 작업 진행 중 결정된 사항

### 텍스처 방식
- 텍스처가 아직 없으므로, fill() 기반으로 시안 테마 색상 적용
- 3D 입체 테두리를 fill()로 구현 (바닐라 스타일: 밝은 쪽 + 어두운 쪽)
- 텍스처 추가 시 blit()으로 교체 가능하도록 ResourceLocation 상수 준비

### HUD 변경
- 이모지 텍스트 → 아이콘 blit() + 텍스트 분리
- 텍스처 미존재 시 텍스트 폴백
