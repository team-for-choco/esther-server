## 작업 진행 중 결정된 사항

### 텍스처 방식
- 텍스처가 아직 없으므로, fill() 기반으로 시안 테마 색상 적용
- 3D 입체 테두리를 fill()로 구현 (바닐라 스타일: 밝은 쪽 + 어두운 쪽)
- 텍스처 추가 시 blit()으로 교체 가능하도록 ResourceLocation 상수 준비

### HUD 변경
- 이모지 텍스트 → 아이콘 blit() + 텍스트 분리
- 텍스처 미존재 시 텍스트 폴백

## 텍스처 생성 프롬프트

Claude.ai에서 아래 프롬프트로 6개 PNG 텍스처를 생성한다.
생성된 파일은 `src/main/resources/assets/estherserver/textures/gui/`에 배치.

```
마인크래프트 모드용 GUI 배경 텍스처를 만들어줘.

규격:
- 픽셀아트 PNG, 투명 배경
- 실제 GUI 영역은 좌상단에 배치, 나머지는 투명

색감 테마 (시안/하늘색 결정체 톤):
- 메인 배경: 어두운 남색 (#2A2A3D)
- 테두리 외곽: 밝은 시안 (#5BC8F5) — 바닐라처럼 3D 입체감 있는 테두리
- 테두리 안쪽: 진한 파랑 (#1A3A5C)
- 내부 패널/슬롯 영역: 짙은 남색 (#1E1E2E)
- 탭 영역: 활성 (#4DB8E8), 비활성 (#3A3A50)
- 전체적으로 바닐라 마인크래프트 GUI의 3D 테두리 스타일을 유지하되,
  회색 대신 파란/시안 톤으로 변환

아래 6개를 각각 개별 PNG 파일로 만들어줘:

0. esther_icon.png (10x10 픽셀, 10x10 캔버스)
   - 에스더의 기운을 상징하는 작은 결정체/보석 아이콘
   - 다이아몬드 형태, 시안~흰색 그라데이션
   - HUD 화폐 표시 옆에 들어갈 작은 심볼

1. collection_bg.png (198x186 영역, 256x256 캔버스)
   - 바닐라 GUI처럼 3D 입체감 테두리 (밝은 시안 외곽 + 진파랑 안쪽)
   - 상단 18px: 타이틀 바 영역 (살짝 밝은 톤)
   - 타이틀 아래 16px: 탭이 들어갈 영역 (가로 줄)
   - 중앙: 9열 격자용 어두운 사각 영역 (162x110px 정도)
   - 하단: 진행률 바 영역

2. shop_bg.png (220x200 영역, 256x256 캔버스)
   - 동일한 3D 테두리 스타일
   - 상단 18px: 타이틀 바
   - 타이틀 아래 16px: 구매/판매 탭 영역
   - 중앙: 4열 격자 영역 (184x140px 어두운 사각)
   - 하단: 잔고 표시 줄

3. profession_bg.png (260x230 영역, 512x256 캔버스)
   - 동일한 3D 테두리
   - 상단 18px: 타이틀 바
   - 4개 직업 행 영역 (각 50px 높이, 240px 폭, 어두운 사각 4줄)
   - 각 행 사이 2px 간격

4. enhancement_bg.png (240x250 영역, 256x256 캔버스)
   - 동일한 3D 테두리
   - 상단 18px: 타이틀 바
   - 4개 장비 행 (각 30px 높이, 220px 폭)
   - 아래: 디테일 패널 영역 (220px 폭 큰 어두운 사각)
   - 최하단: 잔고 표시 줄

5. profession_inventory_bg.png (280x222 영역, 512x256 캔버스)
   - 동일한 3D 테두리
   - 상단 18px: 타이틀 바
   - 탭 영역 (4개, 각 50px 폭)
   - 중앙: 전문 슬롯 영역
   - 하단: 9x4 인벤토리 슬롯 프레임 (각 18x18 슬롯)
```
