# 인챈트 상인 NPC 구현

## 작업 개요

특수 도구 4종의 인챈트 테이블 사용을 차단하고, 화폐로 랜덤 인챈트를 부여하는 새로운 NPC(인챈트 상인)를 구현한다.

CHOCO-143

## 작업 전 요청사항

- 특수 도구 4종(낚싯대/괭이/곡괭이/요리도구) 인챈트 테이블 불가 처리
- 새로운 인챈트 상인 NPC 구현
  - 대상: 방어구 등 기존 인챈트 가능 장비 + 특수 도구 4종 전부
  - 인챈트 종류: 모든 종류 (일반적으로 사용 불가한 것 포함)
  - 비용: 화폐(기운)
  - 두 가지 서비스 모드:
    1. **즉시 부여 (저렴)**: 기존 인챈트에 랜덤 인챈트를 즉시 추가/교체
    2. **선택 부여 (고가)**: 새 랜덤 인챈트를 미리 보여주고 유저가 적용 여부 선택

## 작업 진행 중 결정된 사항

- 특수 낚싯대는 `SpecialFishingRodItem.getEnchantmentValue() = 0` 으로 인챈트 테이블 차단
- 나머지 3종(괭이/곡괭이/요리도구)은 기본 `Item` 이므로 이미 `getEnchantmentValue() = 0`
- `EnchantMerchantEntity`: 별도 엔티티 클래스 (MerchantEntity와 독립)
- 손에 든 아이템을 대상으로 인챈트 적용 (슬롯 인덱스 전달 불필요)
- 비용: 즉시 부여 500기운, 선택 부여 1500기운 (수락 시에만 차감)
- 인챈트는 추가/교체 방식 (같은 타입이면 레벨 교체, 새 타입이면 추가)
- 모든 등록된 인챈트를 랜덤 풀로 사용 (데이터팩 포함)
- 레벨은 해당 인챈트의 최대 레벨 범위에서 랜덤 선택
