## 1. 처음 접속하면

접속하면 **거점(오버월드)** 스폰 지점에 생성됩니다.

### 첫 날 해야 할 것들

1. **특수 장비를 받자** — 관리자 또는 게시판에서 특수 도구 4종을 받아야 서버 콘텐츠를 즐길 수 있습니다. 바닐라 도구로는 커스텀 어종/작물/광물을 획득할 수 없어요.
2. **토지 증서로 내 땅을 만들자** — 처음 접속 시 토지 증서 1개가 지급됩니다. 내 집 자리를 정하고 `/토지` 명령어로 클레임하세요.
3. **마을 게시판에서 퀘스트를 받자** — 게시판 우클릭 한 번으로 일일/주간 퀘스트가 배정됩니다.
4. **야생 차원에서 자원을 모으자** — 오버월드에는 자원이 없습니다. 야생 포탈을 통해 야생 차원으로 가야 합니다.

> **중요**: 레드스톤 계열 아이템은 이 서버에서 비활성화되어 있습니다. 설치도, 조합도, 자연 생성도 없어요.

## 2. 세계 구조

이 서버는 차원이 두 개로 나뉩니다.

### 거점 (오버월드)

- 집을 짓고 생활하는 안전 지대
- **몹이 스폰되지 않습니다** — 야간에도 안전
- **광물이 생성되지 않습니다** — 채굴 불가
- **PvP 비활성화**
- **폭발 피해 없음** (크리퍼, TNT 등)
- 내 땅을 클레임해서 보호할 수 있음

### 야생 차원

- 자원을 캐러 가는 사냥터
- 오버월드와 동일한 지형이지만 **몹 스폰 있음**, **광물 있음**, **PvP 가능**
- 퀘스트의 몬스터 처치는 반드시 **야생 차원**에서만 카운트
- 야생 리셋 기능이 있어서 주기적으로 초기화될 수 있음

### 이동 방법

| 방향 | 방법 |
|------|------|
| 거점 → 야생 | **야생 포탈 블록** 우클릭 |
| 야생 → 거점 | **귀환 포탈 블록** 우클릭 또는 **마을 귀환서** 사용 |

> 마을 귀환서는 어디서든 사용 가능 (야생 차원 포함). 사용 시 3초 카운트다운 후 거점 스폰으로 이동.

## 3. 경제 시스템 — 에스더의 기운

서버 전용 화폐는 **에스더의 기운** (줄여서 '기운'이라고 부릅니다).

- 화면 오른쪽에 HUD로 현재 잔액이 항상 표시됩니다
- 사망해도 잔액은 유지됩니다
- 물리적인 아이템이 아닌 계좌 방식

### 기운 버는 방법

상인 NPC의 **판매 탭**에서 획득한 아이템을 판매하는 것이 기본 수입원입니다.

| 아이템 종류 | 일반 | 고급 | 희귀 |
|------------|------|------|------|
| 어종 (물고기) | 종류별 상이 | 종류별 상이 | 종류별 상이 |
| 수확물 (작물) | 13 | 33 | 73 |
| 광물 원석 | 5 | 13 | 40 |
| 광물 주괴 | 10 | 26 | 80 |
| 레드스톤 | — | — | 250 (드롭 로또) |

퀘스트 보상으로도 기운을 획득할 수 있습니다.

### 기운 쓰는 방법

- 상인 NPC에게서 씨앗, 식재료, 강화 관련 아이템 구매
- 특수 상인에게서 일반 뽑기권, 마을 귀환서 구매

### 플레이어 간 거래

```
/송금 <플레이어명> <금액>
```

## 4. 상인 NPC & 상점

마을에 배치된 NPC에게 우클릭하면 상점 GUI가 열립니다.  
탭으로 **구매**와 **판매**를 전환할 수 있습니다.

### 상인 종류

| 상인 | 구매 가능 항목 |
|------|----------------|
| 씨앗 상인 | 각종 씨앗, 분무기(200), 물뿌리개(500) |
| 식재료 상인 | 요리 재료 (김, 면 등) |
| 광물 상인 | 강화석 (드롭 전용, 상점 구매 불가) |
| 특수 상인 | 일반 뽑기권 (2,000기운), 마을 귀환서 (500기운) |
| 대장장이 NPC | 강화 GUI — 특수 장비 강화 |

> 강화석은 상점에서 살 수 없습니다. 채광(곡괭이 Lv4+) 또는 낚시(낚싯대 Lv4+)로만 획득 가능.

## 5. 핵심 활동: 낚시 / 농사 / 채광 / 요리

이 서버의 주요 콘텐츠는 네 가지 전문 분야입니다.  
모두 **커스텀 아이템**을 사용해야 하며, 바닐라 도구로는 커스텀 콘텐츠를 얻을 수 없습니다.

---

### 낚시

**특수 낚싯대**를 들고 야생 차원의 물가에서 낚시합니다.  
낚싯대 강화 단계에 따라 낚을 수 있는 어종 등급이 달라집니다.

| 낚싯대 강화 | 해금 어종 | 추가 효과 |
|------------|-----------|-----------|
| Lv0 | 커스텀 어종 없음 (바닐라 물고기만) | — |
| Lv1 | 일반 어종 | — |
| Lv2 | 고급 어종 | — |
| Lv3 | 고급 어종 | 5% 확률 연속 낚시 |
| Lv4 | 고급 어종 | + 강화석 낚이는 확률 |
| Lv5 | 고급 어종 | + 물고기가 물면 자동 회수 |

**커스텀 어종 목록** (등급별 예시)

| 등급 | 어종 (일부) |
|------|-----------|
| 일반 | 붕어, 은어, 고등어, 오징어, 멸치, 새우, 조개 … 총 7종 |
| 고급 | 참치, 연어, 도미 … 총 6종 |
| 희귀 | 심해어, 전설 어종 … 총 6종 |

---

### 농사

**특수 호미**로 일반 흙을 **특수 농지**로 변환한 뒤, 씨앗 상인에게 씨앗을 구매해 심습니다.  
호미 강화 단계에 따라 수확 가능한 작물 등급이 달라집니다.

| 호미 강화 | 해금 작물 | 추가 효과 |
|----------|-----------|-----------|
| Lv0 | 커스텀 작물 없음 | — |
| Lv1 | 일반 등급 작물 | — |
| Lv2 | 고급 등급 작물 | — |
| Lv3 | 고급 등급 작물 | 물뿌리개 사용 가능 |
| Lv4 | 고급 등급 작물 | 5% 확률 2개 수확 |
| Lv5 | 희귀 등급 작물 | — |

**분무기 / 물뿌리개**: 작물에 사용하면 성장이 촉진됩니다.  
- 분무기: 3회 충전, 씨앗 상인에서 구매 가능  
- 물뿌리개: 6회 충전, 호미 Lv3 이상 필요, 씨앗 상인에서 구매 가능

**커스텀 작물 목록** (일부)

| 등급 | 작물 |
|------|------|
| 일반 | 쌀, 고추, 시금치, 대파, 마늘, 배추, 콩, 참깨 … |
| 고급 | 6종 |
| 희귀 | 6종 |

---

### 채광

야생 차원에서 **특수 곡괭이**로 커스텀 광물을 채굴합니다.  
곡괭이 강화 단계에 따라 채굴 가능한 광물 등급이 올라갑니다.

| 곡괭이 강화 | 채굴 가능 등급 | 추가 효과 |
|------------|---------------|-----------|
| Lv0 | 나무 등급 광물 | — |
| Lv1 | 돌 등급 광물 | — |
| Lv2 | 철 등급 광물 | — |
| Lv3 | 다이아 등급 광물 | — |
| Lv4 | 다이아 등급 광물 | 강화석 드롭 확률 |
| Lv5 | 다이아 등급 광물 | 강화석 드롭 + **광맥 감지** 파티클 |

**커스텀 광물** (총 11종): 일반 3종 / 고급 4종 / 희귀 4종  
원석 → 제련 → 주괴로 가공하며, 원석보다 주괴가 2배 비쌉니다.

---

### 요리

전용 **조리대** 블록에서 요리합니다. 조리대는 조합이 불가능하며 마을에 설치되어 있습니다.

**요리 방법**

1. **특수 요리 도구**를 들고 조리대에 재료를 하나씩 우클릭으로 투입
2. 모든 재료 투입 완료 후 **빈 손으로 우클릭** → 5초 후 완성
3. 레시피가 맞으면 요리가 완성, 틀리면 재료가 돌아옴

요리 도구 강화에 따라 사용할 수 있는 레시피 등급과 동시에 사용할 수 있는 조리대 수가 늘어납니다.

| 요리 도구 강화 | 해금 레시피 | 동시 조리대 |
|--------------|------------|-------------|
| Lv0 | 일반 레시피 | 1개 |
| Lv1 | 고급 레시피 | 1개 |
| Lv2 | 고급 레시피 | 2개 |
| Lv3 | 고급 레시피 | 4개 |
| Lv4 | 고급 레시피 | 4개 + 5% 확률 2배 완성 |
| Lv5 | 희귀 레시피 | 4개 + 5% 확률 2배 완성 |

**커스텀 요리 목록** (총 24종): 일반 8종 / 고급 8종 / 희귀 8종  
예시: 시금치비빔밥, 생선찌개, 김밥, 김치찌개, 된장국, 고등어구이, 계란밥 …

## 6. 전문 레벨 시스템

네 분야 (낚시/농사/채광/요리) 각각 독립적인 경험치와 레벨(최대 **Lv50**)이 있습니다.

### 경험치 획득

커스텀 아이템을 획득/완성할 때마다 자동으로 해당 분야 XP가 쌓입니다.

| 아이템 등급 | 획득 XP |
|------------|---------|
| 일반 | 1 XP |
| 고급 | 3 XP |
| 희귀 | 5 XP |

### 레벨별 효과

레벨이 오를 때마다 소폭씩 보너스가 누적됩니다.

| 분야 | 레벨당 보너스 | Lv50 최대 |
|------|-------------|-----------|
| 채광 | 채굴 속도 +1% | +50% (1.5배) |
| 낚시 | 낚시 대기시간 -1틱 | -50틱 (약 2.5초) |
| 농사 | 작물 성장 단축 -1% | -50% (절반) |
| 요리 | 요리 시간 단축 -0.05초 | 2.5초 (기본 5초에서) |

### 보관함 해금 마일스톤

| 레벨 | 전문 보관함 슬롯 |
|------|----------------|
| Lv10 | 5칸 해금 |
| Lv20 | +5칸 (총 10칸) |
| Lv30 | +5칸 (총 15칸) |
| Lv40 | +5칸 (총 20칸) |
| Lv50 | +5칸 (총 25칸) |

현황 확인: `/전문기술` 명령어 또는 전문 보관함 GUI(R키)

## 7. 특수 장비 & 강화 시스템

서버 커스텀 콘텐츠를 사용하려면 **특수 도구 4종**이 필수입니다.

| 도구 | 용도 |
|------|------|
| 특수 곡괭이 | 커스텀 광물 채굴 |
| 특수 낚싯대 | 커스텀 어종 낚시 |
| 특수 호미 | 특수 농지 조성 + 커스텀 작물 수확 |
| 특수 요리 도구 | 조리대 요리 |

### 강화 방법

1. 마을의 **대장장이 NPC** 우클릭 → 강화 GUI 열기
2. 특수 장비 + 강화석을 슬롯에 넣으면 강화 시도 (확률 강화)
3. 강화석은 곡괭이 Lv4+ 채광, 낚싯대 Lv4+ 낚시로 획득

### 강화석 드롭 확률

| 획득 방법 | 조건 | 드롭 확률 |
|----------|------|-----------|
| 채광 (일반 광물) | 곡괭이 Lv4+ | 0.5% |
| 채광 (고급 광물) | 곡괭이 Lv4+ | 1% |
| 채광 (희귀 광물) | 곡괭이 Lv4+ | 2% |
| 낚시 | 낚싯대 Lv4+ | 1% |

### 특수 도구 보호

- 사망 시 특수 도구 4종은 **자동으로 보존** (드롭되지 않음, 리스폰 시 복원)
- Q키로 드롭 불가, 상자 등 외부 컨테이너로 이동 불가
- 전문 보관함의 전용 도구 슬롯에 보관 권장

---

## 8. 전문 보관함

**`R키`** 를 누르면 열리는 전용 보관함입니다.

- 4탭 구성: **채광 / 낚시 / 농사 / 요리**
- 각 탭에는 해당 분야 아이템만 보관 가능
- **자동 분류**: 아이템 획득 시 해당 분야 보관함에 우선 저장 (꽉 차면 일반 인벤토리)
- 등급별 테두리 시각 표시 (고급=초록, 희귀=시안)
- 해당 분야 Lv10 이상부터 슬롯이 해금됨 (Lv10: 5칸 ~ Lv50: 25칸)

## 9. 도감 시스템

**`G키`** 를 누르면 열리는 수집 도감입니다.  
아이템을 **도감대 블록**에 우클릭하면 등록됩니다.

### 카테고리

| 카테고리 | 설명 |
|----------|------|
| 어종 | 커스텀 물고기 |
| 작물 | 커스텀 수확물 |
| 광물 | 커스텀 광물/주괴 |
| 요리 | 커스텀 요리 |
| 블록 | 각종 블록 |
| 장비 | 도구/방어구 |
| 음식 | 음식 아이템 |
| 소재 | 기타 재료 |

- 미등록 아이템은 어둡게, 등록 완료 아이템은 밝게 + 체크마크로 표시
- 도감대에 등록 시 파티클/사운드 연출

### 마일스톤 & 보상

일정 수량을 수집하면 마일스톤이 달성되어 서버 전체에 공지됩니다.  
도감 GUI의 **보상 탭**에서 칭호 + 아이템 보상을 수동으로 수령하세요.

총 15개 마일스톤이 있으며, 달성 기준은 10종 → 25종 → 50종 등 단계별로 올라갑니다.

## 11. 퀘스트 시스템

마을에 있는 **게시판** (4×3 멀티블록)을 우클릭해서 퀘스트를 받습니다.

### 기본 구조

| 구분 | 일일 퀘스트 | 주간 퀘스트 |
|------|------------|------------|
| 배정 수 | 5개 | 5개 |
| 최대 완료 | **3개** | **3개** |
| 리셋 주기 | 매일 자정 (KST) | 매주 월요일 자정 (KST) |

### 퀘스트 방식

- **아이템 제출형**: 해당 아이템을 들고 게시판 우클릭 → 자동 제출
- **몬스터 처치형**: 퀘스트 배정 후 야생 차원에서 처치 → 자동 카운트
- 일일 제출 시 주간 퀘스트에도 **동시에 카운트** (아이템 1번만 소모)

### 일일 퀘스트 풀 (분야별 예시)

| 분야 | 예시 퀘스트 | 수량 |
|------|------------|------|
| 낚시 | 붕어 / 은어 / 고등어 / 오징어 / 멸치 / 새우 / 조개 제출 | 3개 |
| 농사 | 쌀 / 고추 / 시금치 / 대파 / 마늘 / 배추 / 콩 / 참깨 제출 | 3개 |
| 채광 | 주석원석 / 아연원석 / 옥원석 제출 | 3개 |
| 요리 | 시금치비빔밥 / 생선찌개 / 김밥 / 김치찌개 등 제출 | 3개 |
| 일반 | 좀비(15) / 스켈레톤(10) / 크리퍼(5) 처치 | 수량 상이 |

### 주간 퀘스트 (고정)

| 분야 | 퀘스트 | 수량 |
|------|--------|------|
| 낚시 | 물고기 아무거나 | 15개 |
| 농사 | 작물 아무거나 | 15개 |
| 채광 | 광물 아무거나 | 15개 |
| 요리 | 요리 아무거나 | 15개 |

### 보상

| 구분 | 기운 | 만년 수프 | 기타 |
|------|------|-----------|------|
| 일일 개별 완료 | 1,000 | 30개 | — |

## 12. 청크 클레임 — 토지 보호

**토지 증서** 아이템을 사용(우클릭)하면 현재 서있는 청크(16×16 블록, 전 높이)가 내 소유가 됩니다.

### 획득 방법

- 최초 접속 시 1개 지급
- 도감 마일스톤 보상
- 특수 상인에서 기운으로 구매 (특수 상인 목록 확인)

### 기본 설정

클레임한 땅은 기본적으로 타인의 **블록 파괴/설치 금지**, **상호작용 허용** 상태입니다.

### 명령어

```
/토지 정보             — 현재 청크 클레임 정보 확인
/토지 목록             — 내 클레임 전체 목록
/토지 제거             — 현재 청크 클레임 해제

/토지 세팅              — 현재 권한 설정 확인
/토지 세팅 파괴 허용|금지    — 블록 파괴 허용/금지
/토지 세팅 설치 허용|금지    — 블록 설치 허용/금지
/토지 세팅 상호작용 허용|금지  — 상자/문/레버 등 허용/금지

/토지 플레이어 추가 <이름>   — 신뢰 플레이어 추가 (내 모든 클레임에 적용)
/토지 플레이어 제거 <이름>   — 신뢰 플레이어 제거
/토지 플레이어 목록         — 신뢰 플레이어 목록
```

> 신뢰 플레이어는 소유자와 동일한 권한을 가집니다.

## 13. 뽑기권 시스템

**일반 뽑기권**을 우클릭하면 CS:GO 스타일 룰렛 애니메이션이 재생된 뒤 결과가 지급됩니다.

### 일반 뽑기권 획득처

- 퀘스트 보상 (일일 3/3 보너스: 1개, 주간 3/3 보너스: 3개)
- 도감 마일스톤 보상
- 특수 상인 구매 (2,000기운)

### 일반 뽑기권 결과

| 결과 | 확률 |
|------|------|
| 펫 뽑기권 | 35% |
| 가구 뽑기권 | 35% |
| 300 기운 | 30% |

### 펫 뽑기권 결과

| 결과 | 비고 |
|------|------|
| 고양이 펫 | 중복 시 500기운으로 전환 |
| 강아지 펫 | 중복 시 500기운으로 전환 |
| 토끼 펫 | 중복 시 500기운으로 전환 |
| 여우 펫 | 중복 시 500기운으로 전환 |

### 가구 뽑기권 결과

- 고양이 소파, 강아지 소파, 토끼 소파, 여우 소파 (2×2×2 멀티블록 가구)

### 인벤토리 세이브권

- 일반 뽑기권에서 약 2% 확률로 드롭
- 사망 시 자동 소모 → 인벤토리, 전문 보관함, 경험치 전부 보존
- 특수 도구는 이미 사망 보존되므로 세이브권과 별개
