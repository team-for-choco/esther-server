# 초반부 완화: 특수 도구 비용 인하 + 낚시 시스템 개선

## 작업 개요

초반 진입 장벽을 낮추고 낚시 콘텐츠에 더 많은 메리트를 부여하기 위한 밸런스 조정.

CHOCO-141

## 작업 전 요청사항

- 특수 도구 0~3레벨 강화 비용 완화
- 낚싯대 더블 낚시 발동 시 숙련도 보너스 추가
- 희귀 어종 낚시 가능 레벨 추가 (Lv5)
- 낚싯대 강화석 드롭을 어종 등급별로 차등 적용

## 작업 진행 중 결정된 사항

- 특수 도구 구매 비용: 5,000 → 1,000
- 4→5 강화: 비용 45,000→31,500, 강화석 1개→5개로 변경
- 희귀 어종 해금 레벨: Lv5 (농작물 희귀 해금 Lv5와 대칭)
- 강화석 드롭 확률: 일반 1% / 고급 2% / 희귀 3%

## 작업 체크리스트

- [x] 특수 도구 구매 비용 인하 (5,000→1,000)
- [x] 강화 비용 인하 (0→1: 500, 1→2: 1,500, 2→3: 4,500)
- [x] 4→5 강화 비용/강화석 개수 조정 (31,500 / 5개)
- [x] EnhancementCost에 stoneCount 필드 추가 및 소모 로직 개선
- [x] 낚싯대 Lv3 더블 낚시 발동 시 보너스 XP 지급
- [x] 낚싯대 Lv5 희귀 어종 해금
- [x] 낚싯대 Lv4 강화석 드롭 등급별 차등 (1%/2%/3%)

## 인게임 테스트

- [ ] 특수 도구 구매 비용 1,000 확인
- [ ] 0→3레벨 강화 비용 인게임 확인
- [ ] 4→5레벨 강화석 5개 요구 확인
- [ ] 낚싯대 Lv5에서 희귀 어종 낚이는지 확인
- [ ] 더블 낚시 발동 시 XP 2회 지급 확인
- [ ] 등급별 강화석 드롭 확률 확인
