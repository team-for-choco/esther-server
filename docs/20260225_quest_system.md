# 일일/주간 퀘스트 시스템

## 작업 개요
플레이어 일일/주간 활동을 유도하고 꾸준한 화폐+음식 수급 경로를 제공하기 위한 퀘스트 시스템 추가.
마을 게시판 블록을 통해 퀘스트를 수령·확인·제출·보상 수령하는 구조.

CHOCO-117

## 작업 전 요청사항
- Phase 7까지 완료된 상태
- 기존 4개 전문 분야(낚시/농사/채광/요리) 연동 필요

---

## 게시판 블록

### 기본 사양
- **멀티블록**: 가로 4 × 세로 3 (총 12블록)
- **설치**: 월드 고정 배치 (조합 불가, OP만 크리에이티브/`/give`로 설치)
- **구조**: 마스터 블록 1개 + 더미 블록 11개, 설치 시 자동 배치
- **상호작용**: 12칸 어디를 우클릭해도 동일하게 동작
- **파괴**: 하나 부수면 전체 제거

### 텍스처
- **독립형** (분수대 앞 광장 배치)
- **앞면**: 나무틀 + 종이/메모가 붙어있는 게시판
- **뒷면**: 나무 판자만 보이는 뒷모습
- **옆면**: 얇은 나무 프레임 두께
- 설치 시 플레이어가 바라보는 방향 기준으로 앞면 결정

---

## 게시판 상호작용 우선순위

게시판 우클릭 시 다음 순서로 처리:

1. **퀘스트 미배정** → 일일+주간 퀘스트 동시 배정
2. **빈 손** → GUI 오픈 (퀘스트 확인/진행도/보상 수령)
3. **아이템 들고** → 해당 아이템 퀘스트에 제출 시도 (일일+주간 동시 카운트)
4. **제출 불가 아이템** → GUI 오픈

### 3개 클레임 시 제출 차단
- 일일/주간 각각 독립적으로 3개 제한
- 일일 3개 클레임 완료 → 나머지 일일 퀘스트 제출/킬 카운트 차단, 주간은 계속 가능
- 주간 3개 클레임 완료 → 나머지 주간 퀘스트 제출/킬 카운트 차단, 일일은 계속 가능
- 양쪽 모두 maxed → 모든 제출 차단, GUI만 열림

## 텍스처 생성 요청 프롬프트

### 공통 지침
```
마인크래프트 모드용 텍스처를 만들어줘.

규격:
- 16x16 픽셀아트
- 투명 배경 (PNG)
- 바닐라 마인크래프트 스타일 (외곽선 있는 픽셀아트, 밝은 하이라이트)
- 테마: 동아시아 (한국 기반 + 일본/중국 요소) 마인크래프트 서버

각 아이템을 개별 16x16 PNG 파일로 만들어줘.
```

### 배치 A: 만년 수프 (1장)
```
마인크래프트 음식 아이템 텍스처 1장을 만들어줘.

1. hunters_pot (만년 수프) — 빨간색 냄비에 빨간 국물 베이스의 음식이 담겨 있는 모습.
   짙은 빨간색/주홍색 냄비(뚝배기), 안에 빨간 국물(김치찌개/짬뽕 느낌).
   윗면에 김이 모락모락 나는 느낌. 바닐라 스튜(mushroom_stew)와 비슷한 레이아웃.
```

### 배치 B: 게시판 블록 (12장 — 위치별 앞면 10장 + 뒷면/옆면 공용)
```
마인크래프트 블록 텍스처를 만들어줘.
마을 광장에 세워두는 나무 게시판 멀티블록이야. 가로 4칸 × 세로 3칸 (총 12블록).

레이아웃 (앞면, 플레이어가 바라보는 방향):
┌──────┬──────────┬──────────┬──────┐
│ 기둥 │ 종이메모 │ 종이메모 │ 기둥 │  ← 3층 (top)
├──────┼──────────┼──────────┼──────┤
│ 기둥 │ 종이메모 │ 종이메모 │ 기둥 │  ← 2층 (mid)
├──────┼──────────┼──────────┼──────┤
│ 기둥 │ (빈공간) │ (빈공간) │ 기둥 │  ← 1층 (bottom)
└──────┴──────────┴──────────┴──────┘

두께 규칙:
- 옆에서 볼 때 양쪽 1/4은 투명, 가운데 1/2(8px)만 나무.
- 앞에서 볼 때 기둥(양 끝 열)은 바깥쪽 절반 투명, 안쪽 절반(8px)만 나무.
- 아래 가운데 2칸은 블록 없음 (기둥 사이 통행 가능).

앞면 텍스처 10장 (위치별 각 1장, 16x16):

1. qb_front_leg_bl — 좌하단 기둥. 왼쪽 절반 투명, 오른쪽 절반에 나무 기둥.
   기둥 하단이 땅에 닿는 모습. 짙은 갈색 나무.
2. qb_front_leg_ml — 좌중단 기둥. 왼쪽 절반 투명, 오른쪽 절반에 나무 기둥.
   세로로 이어지는 나무 기둥 중간 부분.
3. qb_front_leg_tl — 좌상단 기둥. 왼쪽 절반 투명, 오른쪽 절반에 나무 기둥.
   기둥 꼭대기 + 판넬과 연결되는 가로 보 디테일.
4. qb_front_leg_br — 우하단 기둥. 오른쪽 절반 투명, 왼쪽 절반에 나무 기둥.
   (좌하단의 좌우 반전)
5. qb_front_leg_mr — 우중단 기둥. 오른쪽 절반 투명, 왼쪽 절반에 나무 기둥.
6. qb_front_leg_tr — 우상단 기둥. 오른쪽 절반 투명, 왼쪽 절반에 나무 기둥.

7. qb_front_panel_bl — 판넬 좌하단. 나무틀 안에 종이/메모가 붙어있는 모습.
   압정/못 디테일, 작은 글씨/낙서 느낌의 선. 왼쪽·아래 테두리 있음.
8. qb_front_panel_tl — 판넬 좌상단. 위와 비슷하되 위쪽·왼쪽 테두리.
9. qb_front_panel_br — 판넬 우하단. 오른쪽·아래 테두리.
10. qb_front_panel_tr — 판넬 우상단. 오른쪽·위쪽 테두리.

공용 텍스처 2장:

11. qb_back — 뒷면 공용. 기둥 부분은 앞면과 동일한 나무 기둥,
    판넬 부분은 종이 없이 나무 판자만 보이는 뒷모습.
12. qb_side — 옆면 공용. 양쪽 1/4(4px씩) 투명, 가운데 1/2(8px)만 나무.
    판자의 단면이 보이는 얇은 프레임 느낌.
```

### 배치 C: 뽑기권 3종 (3장)
```
마인크래프트 아이템 텍스처 3장을 만들어줘.
뽑기권(티켓) 아이템이야. 약간 구겨진/접힌 종이 티켓 느낌으로,
가운데에 별 또는 보석 마크가 있으면 좋겠어.
등급별로 색상이 달라져야 해.

1. draw_ticket_normal (일반 뽑기권)
   — 흰색/은색 테두리의 티켓. 가운데 흰 별/마크.
   종이 바탕은 밝은 베이지/크림. 수수하고 소박한 느낌.

2. draw_ticket_fine (고급 뽑기권)
   — 초록색 테두리의 티켓. 가운데 초록 보석/별 마크.
   종이 바탕은 밝은 베이지/크림. 약간 빛나는 느낌.

3. draw_ticket_rare (희귀 뽑기권)
   — 시안/하늘색 테두리의 티켓. 가운데 하늘색 보석/별 마크.
   종이 바탕은 밝은 베이지/크림. 맑고 신비로운 느낌.
```
