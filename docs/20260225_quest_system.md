# 일일/주간 퀘스트 시스템

## 작업 개요
플레이어 일일/주간 활동을 유도하고 꾸준한 화폐+음식 수급 경로를 제공하기 위한 퀘스트 시스템 추가.
마을 게시판 블록을 통해 퀘스트를 수령·확인·제출·보상 수령하는 구조.

CHOCO-117

## 작업 전 요청사항
- Phase 7까지 완료된 상태
- 기존 4개 전문 분야(낚시/농사/채광/요리) 연동 필요

---

## 게시판 블록

### 기본 사양
- **멀티블록**: 가로 4 × 세로 3 (총 12블록)
- **설치**: 월드 고정 배치 (조합 불가, OP만 크리에이티브/`/give`로 설치)
- **구조**: 마스터 블록 1개 + 더미 블록 11개, 설치 시 자동 배치
- **상호작용**: 12칸 어디를 우클릭해도 동일하게 동작
- **파괴**: 하나 부수면 전체 제거

### 텍스처
- **독립형** (분수대 앞 광장 배치)
- **앞면**: 나무틀 + 종이/메모가 붙어있는 게시판
- **뒷면**: 나무 판자만 보이는 뒷모습
- **옆면**: 얇은 나무 프레임 두께
- 설치 시 플레이어가 바라보는 방향 기준으로 앞면 결정

---

## 게시판 상호작용 우선순위

게시판 우클릭 시 다음 순서로 처리:

1. **일일 퀘스트 미배정** → 일일 퀘스트 랜덤 배정 (제출 불가)
2. **주간 퀘스트 미배정** → 주간 퀘스트 배정 (제출 불가)
3. **둘 다 배정 + 빈 손** → GUI 오픈 (퀘스트 확인/진행도/보상 수령)
4. **둘 다 배정 + 아이템 들고** → 해당 아이템 퀘스트에 제출 (일일+주간 동시 카운트)

- 퀘스트 배정이 항상 제출보다 우선
- 일일 & 주간 모두 배정된 상태에서만 제출 가능

## 일일 퀘스트

- 게시판 첫 우클릭 시 **플레이어별 개별 랜덤** 배정

## 주간 퀘스트

### 구조
- 이번 주 첫 게시판 우클릭 시 배정
- **고정 5개** (랜덤 없음): 낚시 + 농사 + 채광 + 요리 + 일반
- **최대 3개 완료** 가능
- **매주 월요일 자정 KST 리셋**: 진행도·제출 상태 모두 초기화, 이전 주간 퀘스트 제출 불가

### 퀘스트 목록 (종류 무관 제출/처치)
| 분야 | 퀘스트 | 방식 | 수량 |
|------|--------|------|------|
| 낚시 | 물고기 아무거나 제출 | 제출 | 15 |
| 농사 | 작물 아무거나 제출 | 제출 | 15 |
| 채광 | 광물 아무거나 제출 | 제출 | 15 |
| 요리 | 요리 아무거나 제출 | 제출 | 15 |
| 일반 | 몬스터 아무거나 처치 | 추적 | 30 |

- 일일 퀘스트 제출 시 주간에도 동시 카운트 (아이템 1회 소모로 양쪽 진행)

## 신규 아이템

### 만년 수프 (Hunter's Pot)
- nutrition 8, saturation 0.8f
- 기존 구현 유지

### 뽑기권 (등급별)
- 일반 / 고급 / 희귀 3종 (추후 상위 등급 추가 가능)
- 아이템만 등록 (기능 없음, 추후 구현)
- 등급 색상: 일반=흰색, 고급=초록, 희귀=시안(하늘색)

## 텍스처 생성 요청 프롬프트

### 공통 지침
```
마인크래프트 모드용 텍스처를 만들어줘.

규격:
- 16x16 픽셀아트
- 투명 배경 (PNG)
- 바닐라 마인크래프트 스타일 (외곽선 있는 픽셀아트, 밝은 하이라이트)
- 테마: 동아시아 (한국 기반 + 일본/중국 요소) 마인크래프트 서버

각 아이템을 개별 16x16 PNG 파일로 만들어줘.
```

### 배치 A: 만년 수프 (1장)
```
마인크래프트 음식 아이템 텍스처 1장을 만들어줘.

1. hunters_pot (만년 수프) — 뚝배기/항아리에 담긴 따뜻한 수프. 김이 모락모락 나는 느낌.
   갈색 뚝배기에 황금빛/베이지 국물, 윗면에 작은 파 고명.
   바닐라 스튜(mushroom_stew)와 비슷한 레이아웃이되 더 한국적인 뚝배기 느낌으로.
```

### 배치 B: 게시판 블록 (3장 — 앞면/뒷면/옆면)
```
마인크래프트 블록 텍스처 3장을 만들어줘.
마을 광장에 세워두는 나무 게시판 블록이야.
이 게시판은 멀티블록(가로 4 × 세로 3)으로 조합되지만, 텍스처 자체는 단일 블록(16x16)용이야.

1. quest_board_front (게시판 앞면)
   — 나무 틀(짙은 갈색 테두리) 안에 베이지/흰 종이가 붙어있는 모습.
   종이 위에 작은 글씨/낙서 느낌의 점이나 선이 있으면 좋겠어.
   종이를 고정하는 작은 압정/못 디테일도 추가.
   전체적으로 마을 공지판/게시판 느낌.

2. quest_board_back (게시판 뒷면)
   — 나무 판자만 보이는 뒷모습. 세로 방향 나무결이 보이는 짙은 갈색 판자.
   종이/메모 없이 순수 나무 판자 질감만.

3. quest_board_side (게시판 옆면)
   — 얇은 나무 프레임 두께만 보이는 옆모습. 가운데 2~3픽셀만 나무 색상,
   나머지는 투명. 판자의 단면이 보이는 느낌.
```

### 배치 C: 뽑기권 3종 (3장)
```
마인크래프트 아이템 텍스처 3장을 만들어줘.
뽑기권(티켓) 아이템이야. 약간 구겨진/접힌 종이 티켓 느낌으로,
가운데에 별 또는 보석 마크가 있으면 좋겠어.
등급별로 색상이 달라져야 해.

1. draw_ticket_normal (일반 뽑기권)
   — 흰색/은색 테두리의 티켓. 가운데 흰 별/마크.
   종이 바탕은 밝은 베이지/크림. 수수하고 소박한 느낌.

2. draw_ticket_fine (고급 뽑기권)
   — 초록색 테두리의 티켓. 가운데 초록 보석/별 마크.
   종이 바탕은 밝은 베이지/크림. 약간 빛나는 느낌.

3. draw_ticket_rare (희귀 뽑기권)
   — 시안/하늘색 테두리의 티켓. 가운데 하늘색 보석/별 마크.
   종이 바탕은 밝은 베이지/크림. 맑고 신비로운 느낌.
```

### 파일 배치 경로
```
아이템 텍스처 → src/main/resources/assets/estherserver/textures/item/{이름}.png
블록 텍스처  → src/main/resources/assets/estherserver/textures/block/{이름}.png
```

| B | 게시판 블록 (앞/뒤/옆) | 3 |
| C | 뽑기권 3종 | 3 |
| **합계** | | **7** |
