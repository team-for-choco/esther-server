## CHOCO-98 — 특수 장비 기능화 + 특수 농사용 흙 (Phase 4-1d)
> 원본: [20260219_equipment_farmland.md](20260219_equipment_farmland.md)

**특수 낚싯대:**
- [ ] 투척/회수 정상 동작
- [ ] 생선 낚기 가능
- [ ] 내구도 바 없음, 파괴 안 됨
- [ ] 이 낚싯대로 낚시 시 XP 획득
- [ ] 바닐라 낚싯대로 낚시 시 XP 미획득

**특수 곡괭이:**
- [ ] 돌/광물 채굴 가능
- [ ] requiresCorrectToolForDrops 블록 드롭 정상
- [ ] 내구도 바 없음, 파괴 안 됨
- [ ] 이 곡괭이로 채광 시 XP 획득
- [ ] 바닐라 곡괭이로 채광 시 XP 미획득

**특수 조리도구:**
- [ ] 재료 넣기 정상
- [ ] 조리도구 우클릭 → 요리 완성
- [ ] 맨손 우클릭 → 안내 메시지
- [ ] 요리 XP 정상 부여

**특수 호미 + 커스텀 작물 드롭:**
- [ ] 특수 호미로 성숙 작물 파괴 → 수확물만 드롭
- [ ] 특수 호미로 미성숙 작물 파괴 → 씨앗만 드롭
- [ ] 다른 도구/맨손으로 파괴 → 드롭 없음
- [ ] 바닐라 작물은 정상 드롭
- [ ] 수확 시 농사 XP 획득

**특수 농지 + 수분 시스템:**
- [ ] 상점에서 구매 가능
- [ ] 15px 높이, 기본 건조 상태
- [ ] 밟아도 흙 변환 안 됨
- [ ] 부수면 자기 자신 드롭
- [ ] 건조 상태에서 작물 성장 차단

**분무기:**
- [ ] 상점에서 구매 가능
- [ ] 물에 우클릭 → 3회분 충전
- [ ] 건조 농지에 우클릭 → 비옥화
- [ ] 비옥 농지에 우클릭 → 반응 없음
- [ ] 충전 0일 때 → 안내 메시지
- [ ] 툴팁에 충전량 표시

**농사 게임플레이 루프:**
- [ ] 특수 농지 설치 → 분무기 수분 보충 → 씨앗 심기 → 성장 → 호미 수확
- [ ] 수확 후 농지 건조화
- [ ] 미성숙 작물 제거 시 수분 유지

**커스텀 작물 제한:**
- [ ] 바닐라 farmland에 커스텀 씨앗 심기 불가
- [ ] 특수 농지에 커스텀 씨앗 심기 가능

**XP 게이팅:**
- [ ] 바닐라 도구 → 품질 적용, XP 미획득
- [ ] 특수 장비 → 품질 + XP 모두 획득
