# Phase 5: 콘텐츠 확정 + 비주얼

## 작업 개요
- 테스트 아이템을 실제 아이템으로 교체
- 등급 게이팅이 의미 있도록 분야별 대규모 콘텐츠 추가 (등급당 6종+)
- 테마: 동아시아 믹스 (한국 기반 + 일본/중국 요소)
- 커스텀 텍스처 제작 및 GUI 비주얼 개선

## 작업 전 요청사항
- Phase 4까지 완료 상태
- 텍스처 작업을 위한 도구/방법 결정 필요

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보

### 등급 설계 철학

총 6등급 체계를 염두에 두고 Phase 5에서는 하위 3등급만 구현한다.

| 등급 | 테마 | 획득 난이도 | 예시 |
|------|------|-------------|------|
| **일반 (Common)** | 흔한 일상 소재 | 쉬움 | 붕어, 배추, 주석 |
| **고급 (Advanced)** | 전문가용 특산물 | 보통 | 연어, 표고버섯, 은 |
| **희귀 (Rare)** | 현실 최고급 프리미엄 | 어려움 | 참치, 인삼, 티타늄 |
| *영웅 (Heroic)* | *전설에 한 발 걸친 극상품* | *추후* | *백년 인삼, 흑진주, 다마스쿠스강* |
| *전설 (Legendary)* | *마법이 깃든 자연물* | *추후* | *황금 잉어, 영초, 미스릴* |
| *유물 (Artifact)* | *고대 문명의 유산* | *추후* | *용의 결정, 고대 레시피* |
| *고대 (Ancient)* | *세계의 근원적 힘* | *추후* | *별의 파편, 세계수 열매* |

> **원칙**: 희귀까지는 **현실에 존재하는 것**. 영웅부터 **현실+초자연 경계**, 전설부터 **본격 판타지** 등장.

### 현재 콘텐츠 현황

| 분야 | 현재 | 문제 |
|------|------|------|
| 어종 | 테스트 생선 1종 | 등급 게이팅 무의미 |
| 작물 | 쌀, 고추, 시금치 + 테스트 1종 | 일반 등급만 존재 |
| 광물 | 테스트 광석 1종 | 등급 게이팅 무의미 |
| 요리 | 4종 (모두 일반 등급) | 고급/희귀 레시피 없음 |

## Phase 5-2: 테스트 아이템 교체

기존 "테스트" 아이템을 삭제하고 실제 아이템으로 교체.

| 기존 | 교체 | 처리 |
|------|------|------|
| `test_fish` / `cooked_test_fish` | 삭제, 붕어 등으로 대체 | 루트테이블/도감/가격표 모두 갱신 |
| `test_harvest` / `cooked_test_harvest` | 삭제, 기존 요리 재료 재구성 | 레시피 재료 변경 |
| `test_seeds` / `test_crop` | 삭제 | 블록/아이템/루트테이블 제거 |
| `test_ore` / `deepslate_test_ore` | 삭제, 주석/아연 등으로 대체 | 월드젠/루트테이블 갱신 |
| `test_ore_raw` / `test_ore_ingot` | 삭제 | 위 광석으로 대체 |

**주의**: 기존 월드에 배치된 테스트 블록/아이템은 사라짐. 서버 초기화 또는 마이그레이션 필요.
