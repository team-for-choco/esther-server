## 작업관련 주요 정보

### 1. 특수 곡괭이 채굴 속도 버그 수정
- **문제**: `onBreakSpeed`에서 `BlockTags.MINEABLE_WITH_PICKAXE` 체크 없이 모든 블록에 속도 보너스 적용
- **수정**: 곡괭이로 채굴 가능한 블록에만 속도 보너스 적용
- **추가**: `event.newSpeed` 대신 `event.originalSpeed` 기반 계산으로 다른 모드 호환성 향상

### 2. 분무기 툴팁 다국어 지원
- **문제**: `Component.literal("💧 $charges/$MAX_CHARGES")` 하드코딩
- **수정**: `Component.translatable("tooltip.estherserver.sprayer_charges", charges, MAX_CHARGES)` 사용
- **추가**: ko_kr, en_us 번역 키 추가
