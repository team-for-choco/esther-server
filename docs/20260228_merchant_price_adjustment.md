# 상인 구매/판매 물품 조정 (Phase 11-4)

## 작업 개요
현재 상인 시스템의 구매/판매 가격이 등급별로 일관성이 부족하고, 일부 항목이 누락되어 있음.
기존 가격 체계를 전면 재조정하여 등급별 차등을 명확하게 하고, 구매/판매 비율을 통일한다.

**JIRA**: CHOCO-125

## 작업 전 요청사항
- 강화석 상점 구매: 추가 안 함 (드롭 전용 유지)
- 김/면 구매: 추가 안 함 (현행 유지)
- 커스텀 광물 구매: 추가 안 함 (채광 전용 유지)

## 작업관련 주요 정보

### 설계 원칙
- 구매/판매 비율: 일관되게 ~3x (구매가 = 판매가 x 3)
- 등급 배수: 일반 1x -> 고급 2.5~3x -> 희귀 6~8x
- 가공 보너스: 원석 -> 주괴 2x, 재료 -> 요리 3~5x
- 같은 등급 내 균일화

### 문제점
1. 씨앗 구매/판매 비율 불일치 (4x, 3x, 2.5x 혼재)
2. 같은 등급 내 가격 편차 큼
3. 레드스톤 구매 가능 (CHOCO-105에서 게임 내 차단)
4. 바닐라 씨앗 판매 불가
5. 분무기/물뿌리개 판매 불가

### 수정 파일
- `ShopBuyRegistry.kt` - 구매가 조정
- `ItemPriceRegistry.kt` - 판매가 조정
