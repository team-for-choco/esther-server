# 도감 요리 아이템 툴팁에 재료 개수 힌트 표시

## 작업 개요
요리 레시피 발견이 너무 어렵다는 피드백을 반영하여, 도감에서 미등록 요리 아이템에 마우스를 올리면 해당 레시피의 재료 개수를 힌트로 표시한다.

## 작업 전 요청사항
- 요리가 너무 어렵다는 유저 피드백 (레시피 2개만 발견)
- 레시피 개수(재료 수)만 알아도 큰 도움이 될 것이라는 의견

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보

### 구현 방식
- `CookingStationRecipe`에 `companion object`로 `결과 아이템 → 재료 개수` 정적 캐시 추가
- 레시피는 JSON 로드(서버) 및 네트워크 디코드(클라이언트) 시 생성자가 호출되므로, 양쪽에서 자동으로 캐시 등록
- `CollectionScreen.renderTooltips()`에서 COOKING 카테고리 + 미등록 아이템일 때 "재료 N개" 힌트를 회색 텍스트로 표시
- 이미 등록(수집 완료)한 요리는 힌트 미표시 (이미 알고 있으므로)

### 변경 파일
- `CookingStationRecipe.kt` - 정적 캐시 + `getIngredientCount()` 메서드
- `CollectionScreen.kt` - 툴팁에 힌트 텍스트 추가
- `ko_kr.json` / `en_us.json` - 번역 키 추가
