# 도감 요리 힌트 + 요리 재료 누락 수정

## 작업 개요
요리 레시피 발견이 너무 어렵다는 피드백을 반영하여:
1. 도감에서 미등록 요리 아이템에 마우스를 올리면 재료 개수를 힌트로 표시
2. 요리 재료 중 획득 경로가 누락된 아이템 수정

## 작업 전 요청사항
- 요리가 너무 어렵다는 유저 피드백 (레시피 2개만 발견)
- 레시피 개수(재료 수)만 알아도 큰 도움이 될 것이라는 의견
- 바닐라 아이템 중 요리 재료가 솥에 안 들어간다는 제보 (꿀병)
- 김/면이 획득 경로 없음

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보

### 1. 도감 재료 개수 힌트
- `CookingStationRecipe`에 `companion object`로 `결과 아이템 → 재료 개수` 정적 캐시 추가
- 레시피는 JSON 로드(서버) 및 네트워크 디코드(클라이언트) 시 생성자가 호출되므로, 양쪽에서 자동으로 캐시 등록
- `CollectionScreen.renderTooltips()`에서 COOKING 카테고리 + 미등록 아이템일 때 "재료 N개" 힌트를 회색 텍스트로 표시
- 이미 등록(수집 완료)한 요리는 힌트 미표시 (이미 알고 있으므로)

### 2. 요리 재료 누락 수정
- `minecraft:honey_bottle` (꿀병):
  - `cooking_ingredient` 태그에 추가 (솥 투입 가능)
  - 식재료 상인 구매 15 기운 / 판매 5 기운
  - 유자차 판매가 160 - 재료 원가(73+15+33=121) = 마진 39 기운
- `estherserver:seaweed` (김): 식재료 상인 구매 10 기운
- `estherserver:noodles` (면): 식재료 상인 구매 10 기운
