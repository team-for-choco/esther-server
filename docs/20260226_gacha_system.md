# 뽑기권 시스템 구현

## 작업 개요
2단계 뽑기 구조 프레임워크 구현.
- 일반 뽑기권 → (펫 뽑기권 / 가구 뽑기권 / 화폐)
- 펫 뽑기권 → 일반 등급 펫
- 가구 뽑기권 → 일반 등급 가구

CHOCO-121

## 작업 전 요청사항
- Phase 9(가구) 및 Phase 10(펫) 완료 후 획득 경로 구현
- 퀘스트 보상으로 이미 일반 뽑기권 지급 중

## 작업 진행 중 결정된 사항등, 작업관련 주요 정보

### 구조
```
일반 뽑기권 (우클릭) → 펫 뽑기권(35%) / 가구 뽑기권(35%) / 300 화폐(30%)
펫 뽑기권 (우클릭) → 고양이 펫 (중복 시 500 화폐 전환)
가구 뽑기권 (우클릭) → 고양이 소파
```

### 신규 패키지: `gacha/`
| 파일 | 역할 |
|------|------|
| `GachaTicketItem.kt` | 우클릭 소모 아이템 |
| `GachaRewardPool.kt` | 가중치 보상 풀 |
| `GachaHandler.kt` | 보상 지급 + 중복 펫 처리 + 사운드 |
| `GachaRegistry.kt` | 풀 등록 + 아이템-풀 매핑 |

### 수정 파일
- `EstherServerMod.kt`: 기존 3종 → `GachaTicketItem` 변환, 신규 2종 등록, 크리에이티브 탭, `GachaRegistry.init()`
- `CollectibleRegistry.kt`: 뽑기권 5종 도감 제외
- `ko_kr.json` / `en_us.json`: 신규 번역
- 리소스: items/models/textures 신규 2종

### 중복 펫 화폐 전환 기준
| 펫 등급 | 전환 화폐 |
|---------|----------|
| COMMON | 500 |
| FINE | 1,500 |
| RARE | 5,000 |
