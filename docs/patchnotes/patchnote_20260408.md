### 인챈트 테이블 / 모루 차단

인챈트 테이블과 모루를 통한 인챈트 부여가 차단됩니다.  
모든 인챈트는 **인챈트 상인 NPC**를 통해서만 부여할 수 있습니다.

- **인챈트 테이블**: 아이템을 올려도 인챈트 옵션이 표시되지 않습니다
- **모루**: 인챈트된 책 또는 인챈트된 아이템을 오른쪽 슬롯에 올리면 차단됩니다
  - 이름 변경 및 재료를 이용한 내구도 수리는 기존과 동일하게 가능합니다

### 커스텀 광물 채굴 시 광물 블록 드롭 버그 수정

실크터치가 없는 도구로 커스텀 광물을 채굴할 때 광물 블록이 드롭되던 버그가 수정되었습니다.

- **원인**: 루트 테이블의 실크터치 조건 포맷이 Minecraft 1.21.4와 호환되지 않아, 실크터치 유무에 관계없이 항상 광물 블록이 드롭되던 문제
- **수정 내용**: 커스텀 광물 20종(옥, 오팔, 백금, 루비, 사파이어, 주석, 아연, 티타늄, 탄자나이트, 은 — 일반 / 심층암 각 1쌍) 루트 테이블 포맷을 1.21.4 기준으로 수정
- 실크터치 없이 채굴 시 **원광 드롭**, 실크터치 시 **광물 블록 드롭**이 올바르게 동작합니다
