## 🆕 신규: 바다의 행운 인챈트 효과 확장

**바다의 행운** 인챈트가 커스텀 어종의 출현 확률에도 반영됩니다.

기존에는 바다의 행운이 바닐라 아이템 드롭에만 영향을 주었으나,  
이제 커스텀 어종의 등급별 가중치에도 적용됩니다.

| 인챈트 레벨 | 고급 어종 가중치 | 희귀 어종 가중치 |
|-------------|-----------------|-----------------|
| 없음 | 기본 | 기본 |
| 바다의 행운 Ⅰ | ×1.5 | ×2.0 |
| 바다의 행운 Ⅱ | ×2.0 | ×3.0 |
| 바다의 행운 Ⅲ | ×2.5 | ×4.0 |

> 일반 어종 가중치는 변동 없습니다.

## 📋 유저 가이드 업데이트

유저 가이드 문서의 잘못된 정보가 수정되었습니다.

- **특수 도구 획득**: 관리자 지급 → 거점 대장장이 NPC 구매
- **토지 증서 획득**: 최초 지급 → 도감 1개 등록 보상
- **토지 클레임 방법**: `/토지` 명령어 → 토지 증서 우클릭
- **씨앗 상인 분무기 가격**: 200기운 → 100기운
- **낚싯대 Lv5 해금 어종**: 고급 → 희귀
- **일일 퀘스트 기운 보상**: 1,000기운 → 2,000기운 / 보너스 1,500기운 → 3,000기운
- **일반 뽑기권 결과 목록**: 실제 풀 기준으로 전면 재작성
- **어종 / 광물 목록**: 전체 목록 명시, 고급 어종 참치 오류 수정
