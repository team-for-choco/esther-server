## 🔧 개선

### 일일 퀘스트 재료 등급 제한

일일 퀘스트에 **일반(Common) 등급** 재료만 출현하도록 변경되었습니다.

제거된 항목:
- 낚시: 연어, 갈치, 도미 (고급 등급)
- 요리: 만두, 라면 (고급 등급)

> 기존에 진행 중이던 퀘스트는 정상적으로 완료할 수 있습니다.

### /wild reset 시 네더/엔더 동시 초기화

야생 초기화 명령 실행 시 **네더**와 **엔드** 차원도 함께 리셋됩니다.  
해당 차원에 있는 플레이어는 자동으로 거점(오버월드)으로 대피됩니다.

### 야생 자동 리셋

매주 **월요일 00시(KST)** 에 야생이 자동으로 리셋됩니다.  
별도의 관리자 명령 없이 서버가 자동으로 처리합니다.

### 토지 권한 상속

`/토지 세팅`으로 변경한 권한(설치/파괴/상호작용)이 **새로 설치하는 청크에도 자동 적용**됩니다.  
기존에는 새 청크마다 기본값으로 생성되어 매번 다시 설정해야 했습니다.

## 🐛 버그 수정

### 타인 토지 블록 설치 시 아이템 미반환

다른 플레이어의 토지에 횃불 등 블록을 설치했을 때, 블록은 사라지지만 아이템이 즉시 돌아오지 않던 문제가 수정되었습니다.  
(기존에는 재접속해야 복구)

### 가구(소파) 파괴 시 아이템 미드롭

여우/개/토끼 소파를 파괴했을 때 아이템이 드롭되지 않고 사라지던 문제가 수정되었습니다.  
(고양이 소파는 정상이었으나, 나머지 3종에 드롭 테이블이 누락되어 있었음)

### 강화석 요구량 표시 오류

4→5 강화 시 강화석이 "x1"로 표시되던 문제가 수정되었습니다.  
실제 요구량인 **x5**가 정상적으로 표시되며, 보유 수량이 부족하면 강화 버튼이 비활성화됩니다.

### 전문 분야 현황(J키) 장비 미인식

전문 보관함에 넣어둔 도구가 "장비 미보유"로 표시되던 문제가 수정되었습니다.  
이제 일반 인벤토리, 오프핸드, 전문 보관함 모두에서 장비를 인식합니다.
